 +---------------------+
 | [X] Force Far Calls |
 +---------------------+

[X] When Force Far Calls is on (checked), all
procedures and functions use the FAR call
model.

[ ] If Force Far Calls is not checked, the
compiler uses the NEAR call models for any
procedures or functions within the file being
compiled.

Force Far Calls is equivalent to {$F}.
