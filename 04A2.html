 MoveTo (procedure)     (Graph unit)
 ------------------------------------
Moves the current pointer (CP) to (X,Y).

 Declaration:
 procedure MoveTo(X, Y: Integer);

 Target:
Real, Protected

 Remarks:
The CP is similar to a text mode cursor except that the CP is not visible.
The following routines move the CP:

 ClearDevice
 ClearViewPort
 GraphDefaults
 InitGraph
 LineRel
 LineTo
 MoveRel
 MoveTo
 OutText
 SetGraphMode
 SetViewPort

If a viewport is active, the CP will be viewport-relative (the X and Y
values will be added to the viewport's X1 and Y1 values). The CP is never
clipped at the current viewport's boundaries.

 See Also:
 GetMaxX
 GetMaxY
 GetX
 GetY
 MoveRel

 Sample Code:



 {Moveto.PAS}

 {Sample code for the MoveTo procedure.}

 uses Graph;

 var Gd, Gm: Integer;
 begin
  Gd := Detect;
  InitGraph(Gd, Gm, ' ');
  if GraphResult <> grOk then
    Halt(1);
  MoveTo(0, 0); { Upper left corner of viewport }
  LineTo(GetMaxX, GetMaxY);
  Readln;
  CloseGraph;
 end.
