 +--------------------------+
 | [X] EGA/VGA Palette Save |
 +--------------------------+

[X] When you check EGA/VGA Palette Save (and
click OK in the Startup Options dialog box),
the IDE restores the EGA palette each time
the screen is swapped (after you restart it).

You generally don't need to use this option
unless your program modifies the EGA palette
registers or uses BGI to change the palette.

 +-----------------------------------+
 | Default setting: Off ([ ] or /P-) |
 | Command-line equivalent: /P       |
 +-----------------------------------+
