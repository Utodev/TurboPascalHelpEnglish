  /C Command-Line Option (BPC only)
 -----------------------------------
Use the /C command-line option to choose the target platform.

   Option | Platform
   =======================================
   /CD    | DOS real mode (default option)
   /CP    | DOS protected mode
   /CW    | Windows

The /C option controls which target platform the $M option will affect. The
/C option must appear before $M on the command line.
