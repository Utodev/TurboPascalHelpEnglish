 +----------------------+
 | [X] Emulation        |
 +----------------------+

[X] When you check Emulation, Turbo Pascal
links with software routines that use an 80x87
if one is present, or if there is no 80x87, the
run-time library emulates it.

[ ] When Emulation is not checked, the compiler generates
code that can be used only if an 80x87 is present.

This option is not available if Target platform options is set to Windows.

Emulation is equivalent to {$E}.

See also:
