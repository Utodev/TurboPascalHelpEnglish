 - Help on the Add Breakpoints dialog box
 -----------------------------------------
The Add Breakpoints dialog box is where you add breakpoints.
Whenever your running program encounters a breakpoint, it
will stop with a run bar positioned on the line with the breakpoint.

 +- Condition --------------------------+
 |                                      |
 +--------------------------------------+

The Condition input box is where you enter the
condition that causes the breakpoint to break.

 +- Pass Count -------------------------+
 |                                      |
 +--------------------------------------+

In Pass Count, you set how many times the
breakpoint should be skipped before execution
stops.

 +- File Name --------------------------+
 |                                      |
 +--------------------------------------+

The File Name input box is where you enter the full path name of the
source file containing the current breakpoint.

 +- Line Number ------------------------+
 |                                      |
 +--------------------------------------+

The Line Number input box is where you enter the line number of the
breakpoint.
