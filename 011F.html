 | [X] Integrated  |
 +-----------------+

[X] Programs that are linked with Integrated
checked (on) can be debugged with either the
integrated debugger or with Turbo Debugger.

This is the default setting.

If you choose Integrated, do it before you
compile and link your program.

The Integrated option produces complete
debugging information for a module only
if that module has been compiled with the
Options|Compiler Debug Information option checked.

Integrated produces local symbol information
for a module only if that module has been
compiled with the O|C Local Symbols option
checked.
