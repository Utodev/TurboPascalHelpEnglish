 +--------------------------+
 | [X] CGA Snow Checking    |
 +--------------------------+

[X] When you check CGA Snow Checking (and
click OK in the Startup Options dialog box),
the IDE performs CGA snow checking the next
time you start TURBO.EXE, TPX.EXE or BP.EXE.

This option has an effect only if you're
using a CGA.

[ ] Disable (uncheck) CGA Snow Checking if
you're using a CGA that doesn't experience
snow during screen updates.

 +-----------------------------------+
 | Default setting: On ([X] or /N+)  |
 | Command-line equivalent: /N       |
 +-----------------------------------+
