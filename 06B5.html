  Disk status functions
 ----------------------

 Disk status functions | What it does
 ===========================================================================
 DiskFree (function)   | Returns the number of free bytes of a specified
                       | disk drive.
 DiskSize (function)   | Returns the total size, in bytes, of a specified
                       | disk drive.
 GetVerify (procedure) | Returns the state of the verify flag in DOS.
 SetVerify (procedure) | Sets the state of the verify flag in DOS.
