 Browsing Globals
 -----------------
The Search|Global command opens a window that displays the global
symbols used in your program listed in alphabetical order.

Select a symbol and press Enter or double-click the symbol to browse.

You can browse globals in the same ways you can browse objects.

See Using the ObjectBrowser for more information.
