  $SAVE ALL, $SAVE CUR, and $SAVE PROMPT
 -----------------------------------------
The $SAVE ALL, $SAVE CUR, and $SAVE PROMPT macros tell the IDE to

  - save all files, or
  - save the file in the current edit window, or
  - prompt you when there are unsaved files

   $SAVE ALL            The $SAVE ALL macro tells the IDE to save all files
                        in all edit windows that have been modified, without
                        prompting.
   $SAVE CUR            The $SAVE CUR macro tells the IDE to save the file
                        in the current edit window if it has been modified.
                        This ensures that the transfer program will use the
                        latest version of the source file.
   $SAVE PROMPT         The $SAVE PROMPT macro tells the IDE to prompt when
                        there are unsaved files in edit windows.
