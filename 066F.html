  Overlay Unit
 ----------------

The Overlay unit (OVERLAY.TPU) contains procedures, functions, and symbols
used with Borland Pascal's overlay manager.

 Target:
Real mode only

 Functions and Procedures :
 OvrClearBuf
 OvrGetBuf
 OvrGetRetry
 OvrInit
 OvrInitEMS
 OvrSetBuf
 OvrSetRetry

 Constants, Types, and Variables :
 type OvrReadFunc = function(OvrSeg : Word): Integer;

 ovrError         = -1;
 ovrNotFound      = -2;
 ovrNoMemory      = -3;
 ovrIOError       = -4;
 ovrNoEMSDriver   = -5;
 ovrNoEMSMemory   = -6;
