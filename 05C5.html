 Set-type constants
 --------------------
The declaration of a set-type constant specifies the value of the set using
a constant expression.

 Examples:
 type
   Digits = set of 0..9;
   Letters = set of 'A'..'Z';
 const
   EvenDigits: Digits = [0, 2, 4, 6, 8];
   Vowels: Letters = ['A', 'E', 'I', 'O', 'U', 'Y'];
   HexDigits: set of '0'..'z' = ['0'..'9', 'A'..'F', 'a'...f'];
