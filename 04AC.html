 OvrClearBuf (procedure)     (Overlay unit)
 -------------------------------------------
Clears the overlay buffer.

 Declaration:
 procedure OvrClearBuf;

 Target:
 Real

 Remarks:
Disposes of all currently loaded overlays from the overlay buffer. This
forces subsequent calls to overlaid routines to reload the overlays from
the overlay file (or from EMS). If OvrClearBuf is called from an overlay,
that overlay will immediately be reloaded upon return from OvrClearBuf.

The overlay manager never requires you to call OvrClearBuf; in fact, doing
so will decrease performance of your application, because it forces
overlays to be reloaded. OvrClearBuf is solely intended for special use,
such as temporarily reclaiming the memory occupied by the overlay buffer.

 See Also:
 OvrGetBuf
 OvrSetBuf

 Sample Code:



  {OvrClrbf.PAS}

 { Example for OvrClearBuf: Runs only in DOS real mode }

 {$M 16384,65536,655360}
 uses Overlay;
 begin
   OvrInit('EDITOR.OVR');
   if OvrResult <> ovrOk then
     Halt(1);
   WriteLn('We are going to use the overlay ',
           'buffer as some extra temporary ',
           'storage.');
   OvrClearBuf; { Flush the overlay buffer }
   { Non-overlaid code can use the overlay
     buffer for other purposes until the
     next call to an overlaid subprogram.
     The available area is OvrHeapOrg:0 to
     OvrHeapEnd:0 (Segment:Offset). }
 end.
