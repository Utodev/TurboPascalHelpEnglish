 StrComp (function)     (Strings unit)
 --------------------------------------
Compares two strings.

 Declaration:
 function StrComp(Str1, Str2 : PChar): Integer;

 Target:
Windows, Real, Protected

 Return Value:
 <0        if Str1 < Str2
 =0        if Str1 = Str2
 >0        if Str1 > Str2

 Remarks:
StrComp compares Str1 to Str2. The return value is less than 0 if
Str1 < Str2, 0 if Str1 = Str2, or greater than 0 if Str1 > Str2.

 See Also:
 StrIComp
 StrLComp
 StrLIComp

 Sample Code:



 {StrComp.PAS}

 {Sample code for the StrComp and StrlComp functions.}

 { For Windows: }
 { uses Strings, WinCrt; }

 uses Strings;

 var
  C: Integer;
  Result: PChar;
  S1, S2: array[0..79] of Char;
 begin
  Readln(S1);
  Readln(S2);
  C := StrComp(S1, S2);
  if C < 0 then Result := ' is less than ' else
    if C > 0 then Result := ' is greater than ' else
      Result := ' is equal to ';
  Writeln(S1, Result, S2);
 end.
