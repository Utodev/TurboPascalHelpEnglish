 - Search|Globals (BP and BPW)
 ------------------------------
The Search|Globals command displays a browser window with
global identifier information displayed.

Use the arrow keys to move around the window. To inspect an identifier,
highlight it and press Enter or click directly on the identifier name.
You can then go to the source code that defines the highlighted
identifier or inspect the declaration of the highlighted identifier.

See Using ObjectBrowser for more details on using the ObjectBrowser.
