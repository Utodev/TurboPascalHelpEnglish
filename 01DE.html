 - Debug|User Screen     (Alt+F5)
 ---------------------
Choose User Screen to view your program's
full-screen output.

You can view both text and graphics output at
the User Screen; the Output window only
displays text output.

If you would rather see your program output in
a Turbo Pascal window inside the environment,
choose the Debug|Output command instead.

Click or press any key to return to the Turbo
Pascal environment.
