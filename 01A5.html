 Turbo Pascal's Standard Units
 ------------------------------
A unit is a collection of constants, data types, variables, procedures, and
functions.

Turbo Pascal provides the following standard units:
 Crt
 Dos
 Graph
 Graph3
 Overlay
 Printer
 System
 Turbo3
 WinDos
These standard units provide support for your Turbo Pascal programs and
are stored in TURBO.TPL (DOS real mode).

Each unit is like a separate Pascal program: It can have a main body that is
called before your program starts and does whatever initialization is
necessary.
