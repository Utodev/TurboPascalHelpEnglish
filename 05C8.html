  Structured-type constants
 ---------------------------
The declaration of a structured-type constant specifies the value of each of
the structure's components.

Borland Pascal supports the declaration of type array, record, set, and
pointer constants.
Type file constants and constants of array and record types that contain
type file components are not allowed.
