 StrCopy (function)     (Strings unit)
 --------------------------------------
Copies one string to another.

 Declaration:
 function StrCopy(Dest, Source: PChar): PChar;

 Target:
Windows, Real, Protected

 Remarks:
StrCopy does not perform any length checking.
The destination buffer must have room for at least StrLen(Source)+1
characters.

 See Also:
 StrECopy
 StrLCopy

 Sample Code:



 {StrCopy.PAS}

 {Sample code for the StrCopy function.}

 { For Windows: }
 { uses Strings, WinCrt; }

 uses Strings;

 var
  S: array[0..15] of Char;
 begin
  StrCopy(S, 'Turbo Pascal');
  Writeln(S);
 end.
