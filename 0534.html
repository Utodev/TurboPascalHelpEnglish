 ClearViewPort (procedure)     (Graph unit)
 -------------------------------------------
Clears the current view port (graphics mode only).

 Declaration:
 procedure ClearViewPort;

 Target:
Real, Protected

 Remarks:
ClearViewPort sets the fill color to the background color (Palette[0]) and
moves the current pointer to (0, 0).

 Restrictions:
Must be in graphics mode.

 See Also:
 ClearDevice
 GetViewSettings
 SetViewPort

 Sample Code:



 {Clrvwprt.PAS}

 {Sample code for the ClearViewPort procedure.}

 uses Graph;

 var Gd, Gm: Integer;
 begin
  Gd := Detect;
  InitGraph(Gd, Gm, ' ');
  if GraphResult <> grOk then
    Halt(1);
  Rectangle(19, 19, GetMaxX - 19, GetMaxY - 19);
  SetViewPort(20, 20, GetMaxX - 20, GetMaxY - 20, ClipOn);
  OutTextXY(0, 0, ' clears viewport:');
  Readln;
  ClearViewPort;
  OutTextXY(0, 0, ' to quit:');
  Readln;  CloseGraph;
 end.
