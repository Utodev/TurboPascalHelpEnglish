 | [Chdir ] |
 +----------+

Once you've typed a directory name in the
input box, or selected one in the Directory
Tree list box, choose the Chdir button.

Choose Cancel if you change your mind.

You can also just press Enter once the file
is selected, or you can double-click the
file name.
