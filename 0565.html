 Frac (function)
 ----------------
Returns the fractional part of the argument.

 Declaration:
 function Frac(X: Real): Real;

 Target:
Windows, Real, Protected

 Remarks:
X is a real-type expression. The result is the fractional part of X; that
 is, Frac(X) = X - Int(_X).

 See Also:
 Int

 Sample Code:



 {Frac.PAS}

 {Sample code for the Frac function.}

 var
  R: Real;
 begin
  R := Frac(123456);    { 0.456 }
  R := Frac(-123456);   { -0.456 }
 end.
