 - Help on the Mouse Options dialog box
 ---------------------------------------
This dialog box consists of one set of radio
buttons, one check box, one slider bar, and
the standard buttons OK, Cancel, and Help.

You use the settings in this dialog box to
customize the Control+right mouse button actions in the Turbo
Pascal environment.

 +----------------------------+
 | Control+Right Mouse Button |
 |  ( ) Nothing               |
 |  (.) Topic search          |
 |  ( ) Go to cursor          |
 |  ( ) Breakpoint            |
 |  ( ) Evaluate              |
 |  ( ) Add watch             |
 |  ( ) Browse symbol         |
 +----------------------------+
The Control+Right Mouse Button radio buttons determine
what happens when you press Control+right (not
left) mouse button.

Right mouse button always brings up a local menu.

 +----------------------------+
 | Mouse Double Click         |
 | Fast     Medium     Slow   |
 | ±±±±±±±±±±±»±±±±±±±±±±±±±|
 +----------------------------+
The Mouse Double Click slider bar adjusts the
double-click speed of your mouse.

 +---------------------------+
 | [ ] Reverse Mouse Buttons |
 +---------------------------+
Reverse Mouse Buttons determines which of
your mouse buttons is the active one.
