 - Options|Environment|Startup...
 ---------------------------------
Startup brings up the Startup Options dialog
box, where you select settings for the IDE.

The changes you make in the Startup Options
dialog box are written directly to TURBO.EXE
and don't take effect until the next time you
load the IDE.
