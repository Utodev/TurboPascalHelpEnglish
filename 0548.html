 DoneWinCrt (procedure)     (WinCrt unit)
 -----------------------------------------
Destroys the CRT window if it has not already been destroyed.

 Declaration:
 procedure DoneWinCrt;

 Target:
Windows

 Remarks:
Call DoneWinCrt just before the program ends to prevent the CRT window from
entering the inactive state.

 Sample Code:



  {DoneWCrt.PAS}

 {Sample code for the DoneWinCrt procedure.}

 uses WinCrt;

 begin
   Writeln('This is the first CRT window.');
   Writeln('  Please hit return.');
   Readln;
   DoneWinCRT;
   Writeln('This is the second CRT window.');
 end.
