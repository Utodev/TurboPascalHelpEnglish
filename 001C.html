 - Debug     (Alt+D)
 --------
The commands on the Debug menu control all the
features of the integrated debugger.

You can change default settings for these
commands in the Options|Debugger dialog box.

These are the available menu commands:

 Breakpoints        
 Call stack         
 Register           
 Watch              
 Output             
 User screen        
 Evaluate/Modify    
 Add Watch          
 Add Breakpoint
