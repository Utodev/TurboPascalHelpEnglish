 Hi (function)
 --------------
Returns the high-order byte of the argument.

 Declaration:
 function Hi(X): Byte;

 Target:
Windows, Real, Protected

 Remarks:
X is an expression of type Integer or Word. Hi returns the high-order byte
of X as an unsigned value.

 See Also:
 Lo
 Swap

 Sample Code:



 {Hi.PAS}

 {Sample code for the Hi function.}

 var B: Byte;
 begin
  B := Hi($1234);   { $12 }
 end.
