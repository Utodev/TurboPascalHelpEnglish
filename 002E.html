 - Edit|Redo
 ------------
The Redo command takes back the last
Undo command you performed on a line.

It can also redo a Ctrl-Y, or text typed
on a blank line.

Redo works only on the Undo of a modified
or deleted line. However, you can press Redo
to keep "redoing" previous changes.

Set Group Undo in the
Options| Environment|Editor Editor Options
dialog box to Undo (or Redo) more than a single
keystroke.
