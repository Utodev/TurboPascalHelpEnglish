 SetCurDir (procedure)     (WinDos unit)
 ----------------------------------------
Changes the current directory to the specified path.

 Declaration:
 procedure SetCurDir(Dir: PChar)

 Target:
Windows

 Remarks:
If Dir specifies a drive letter, tthe drive is also changed.

 See Also:
 GetCurDir
 ChDir
 CreateDir
 RemoveDir

 Sample Code:



 {SetCurDr.PAS}

 {Sample code for the SetCurDir procedure.}

 { For Windows: }
 { uses WinCrt, WinDos; }

 uses WinDos;

 const
  ChangeTo: PChar = 'C:\';
  CDrive: Byte = 3;

 var
  CurDir: PChar;

 begin
  GetMem(CurDir, 80);
  SetCurDir(ChangeTo);
  GetCurDir(CurDir, CDrive);
  Writeln('The current directory on drive C is ', CurDir,'.');
 end.
