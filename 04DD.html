 SetFAttr (procedure)     (Dos unit and WinDos unit)
 ----------------------------------------------------
Sets the attributes of a file.

 Declaration:
 procedure SetFAttr(var F; Attr: Word);

 Target:
Windows, Real, Protected

 Remarks:
Errors are reported in DosError. Possible error codes are
 -  3 (Invalid Path)
 -  5 (File Access Denied)

 Restrictions:
The file cannot be open.

 See Also:
 GetFAttr
 GetFTime
 SetFTime

 Sample Code:



  {Setfattr.PAS}

 {Sample code for the SetFAttr procedure.}

 { For Windows: }
 { uses WinDos, WinCrt; }

 uses Dos;

 var
   F: file;
 begin
   Assign(F, 'C:\AUTOEXEC.BAT');
   SetFAttr(F, Hidden);                { For Windows: faHidden }
   Readln;
   SetFAttr(F, Archive);          { For Windows: faArchive }
 end.
