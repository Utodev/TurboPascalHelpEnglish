 StrUpper (function)     (Strings unit)
 ---------------------------------------
Converts a string to upper case.

 Declaration:
 function StrUpper(Str: PChar): PChar;

 Target:
Windows, Real, Protected

 Remarks:
StrUpper converts Str to uppercase and returns Str.

 See Also:
 StrLower

 Sample Code:



 {StrUpper.PAS}

 {Sample code for the StrUpper function.}

 { For Windows: }
 { uses Strings, WinCrt; }

 uses Strings;

 var
  S: array[0..79] of Char;
 begin
  Readln(S);
  Writeln(StrUpper(S));
  Writeln(StrLower(S));
 end.
