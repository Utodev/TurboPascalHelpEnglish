 - Options|Environment|Mouse...
 -------------------------------
The Mouse command leads to the Mouse Options
dialog box, where you set various options
that control how your mouse works, including:

 » how fast a double-click is
 » which mouse button (right or left) is
   used to select options, mark blocks, etc.
 » which function or procedure (if any) is
   tied to the Ctrl+Right mouse button
