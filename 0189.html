 - Colors
 --------
You use the Colors set of radio buttons on
monochrome systems to modify the character
attributes of IDE views.

 +----------------------+
 |  (x) Mono Low        |
 +----------------------+
The Mono Low option sets the attribute of the
currently selected IDE item (view) in the
currently selected group to low intensity
("normal").

 +----------------------+
 |  ( ) Mono High       |
 +----------------------+
The Mono High option sets the attribute of
the currently selected IDE item in the
currently selected group to high intensity.

 +----------------------+
 |  ( ) Mono Underscore |
 +----------------------+
The Mono Underscore option sets the attribute
of the currently selected IDE item in the
current group to underscore.

 +----------------------+
 |  ( ) Mono Inverse    |
 +----------------------+
The Mono Inverse option sets the attribute of
the current IDE item in the current group to
inverse ("reverse video").

 +----------------+
 | Text Text Text |
 | Text Text Text |
 +----------------+
The block of display text below the radio
buttons shows the current attribute
setting.
