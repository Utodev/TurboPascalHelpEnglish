 Color-Related Procedures and Functions, 3.0 Graphics
 -----------------------------------------------------
 Procedure       | What It Does
 ============================================================================
 ColorTable      | Defines a color translation table. With this table, the
                 | current color of any given point determines the new color
                 | of that point when it is redrawn
 GraphBackground | Sets background color of screen
 HiResColor      | Selects the color used for drawing in high-resolution
                 | graphics
 Palette         | Activates the color palette specified
