 P (Pointer format specifier)
 -----------------------------
Displays pointers in segment:offset format, with additional information
about the address pointed to, rather than in the default hardware-oriented
Ptr(seg,ofs) format.

Specifically, the P format specifier tells you the region of memory in
which the segment is located, and (if appropriate) the name of the variable
at the offset address.

Affects only pointer values.
