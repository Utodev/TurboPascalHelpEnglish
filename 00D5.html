 +- Parameters -------------------------+
 |                                      |
 +--------------------------------------+

This is where you enter the parameters that
you want Turbo Pascal to pass to your running
program.

Parameters take effect only when your program
is started.

You don't need to enter the program name in
this input box.

You can use the history list that's attached
to the Parameters input box to automatically
re-enter parameters you've previously typed.

If you're already debugging and want to change
parameters, you can select Run|Program Reset to
start the program with the new parameters.
