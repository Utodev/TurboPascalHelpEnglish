 StrLower (function)     (Strings unit)
 ---------------------------------------
Converts a string to lower case.

 Declaration:
 function StrLower(Str: PChar): PChar;

 Target:
Windows, Real, Protected

 Remarks:
StrLower converts Str to lowercase and returns Str.

 See Also:
 StrUpper

 Sample Code:



 {StrUpper.PAS}

 {Sample code for the StrUpper function.}

 { For Windows: }
 { uses Strings, WinCrt; }

 uses Strings;

 var
  S: array[0..79] of Char;
 begin
  Readln(S);
  Writeln(StrUpper(S));
  Writeln(StrLower(S));
 end.
