 MAKE utility
 ------------
Use MAKE to create real-mode executables compiled
by BPC or TPC.

Syntax:
 make [option] [target]

 option is a make option
 target is a filename

Each make option must be separated by a space. Options can
be placed in any order.

You can write implicit and explicit rules or macros.

See the UTILS.DOC for complete information.
