 | [X] 8087/80287       |
 +----------------------+

[X] When you check 8087/80287, the compiler
generates direct 8087 or 80287 inline code for
handling floating-point numbers.

If 8087/8287 is not selected, the compiler calls
run-time library routines to perform all real-type
calculations.

8087/80287 is equivalent to {$N}.
