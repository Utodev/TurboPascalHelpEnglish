 - Help on the ^K W Write Block dialog box
 ------------------------------------------

The ^K W Write Block dialog box is where you
save a selected block of text to a file.

The ^K W Write Block dialog box contains an
input box, a file list, a file information
panel, and the standard OK, Cancel and Help
buttons, plus a history list.

 +- Write block to ----------------+
 |                                 |
 +---------------------------------+
The Write Block To input box is where you
enter the file name you want to save the
block under, or the file-name mask for the
Files list box.

 + Files --------------------------+
 |                                 |
 |  FILENM01.PAS  |  FILENM09.PAS  |
 |  FILENM02.PAS  |  FILENM10.PAS  |
 |  FILENM03.PAS  |  FILENM11.PAS  |
 |  FILENM04.PAS  |  FILENM12.PAS  |
 |  FILENM05.PAS  |  ..            |
 |  FILENM06.PAS  |  \MOREXAMP     |
 |  FILENM07.PAS  |  \TOURS        |
 |  FILENM08.PAS  |  \ANSWERS.DIR  |
 |  ±±±»±±±±±±±±±±±±±±±±±±±±±±±± |
 +---------------------------------+
The Files list box lists the names of files in
the current directory that match the file-name
mask in the Write Block To input box, plus the
parent directory and all subdirectories.

 +------------------------------------------+
 | C:\TP\EXAMPLES\*.PAS                     |
 | HELLO.PAS       52  May 14,1992  12:00pm |
 +------------------------------------------+
The File information panel shows the path
name, file name, date, time, and size of the
selected file.
