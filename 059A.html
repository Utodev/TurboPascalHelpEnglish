 Include (procedure)
 --------------------
Includes an element in a set.

 Declaration:
 procedure Include(var S: set of T; I:T);

 Target:
Windows, Real, Protected

 Remarks:
S is a set type variable, and I is an expression of a type compatible with
the base type of S. The element given by I is included in the set given by
S.

The construct Include (S,I) corresponds to S := S + (I) but the Include
procedure generates more efficient code.
