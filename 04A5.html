 NormVideo (procedure)     (Crt unit)
 -------------------------------------
Selects the original text attribute read from the cursor location at
startup.

 Declaration:
 procedure NormVideo;

 Target:
Real, Protected

 Remarks:
There is a Byte variable in Crt--TextAttr--that holds the current video
attribute. NormVideo restores TextAttr to the value it had when the program
was started.

 See Also:
 HighVideo
 LowVideo
 TextBackground
 TextColor

 Sample Code:



  {NormVid.PAS}

 { Example for NormVideo, TextBackGround,
   and TextColor }

 uses Crt;
 begin
   { Green characters on black}
   TextColor(Green);
   TextBackground(Black);
   WriteLn('Hey there!');
   { Blinking light-red characters }
   { on gray }
   TextColor(LightRed+Blink);
   TextBackground(LightGray);
   WriteLn('Hi there!');
   { Yellow characters on blue }
   TextColor(14); { Yellow = 14 }
   TextBackground(Blue);
   WriteLn('Ho there!');
   NormVideo; { Original attribute }
   WriteLn('Back to normal...');
 end.
