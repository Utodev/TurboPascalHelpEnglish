 IOResult (function)
 --------------------
Returns the status of the last I/O operation performed.

 Declaration:
 function IOResult: Integer;

 Target:
Windows, Real, Protected

 Return Value:
0 if successful.

 Remarks:
I/O-checking must be off--{$I-}--to trap I/O errors using IOResult. If an
I/O error occurs and I/O-checking is off, all subsequent I/O operations are
ignored until a call is made to IOResult. A call to IOResult clears the
internal error flag.

 Remarks:
A call to IOResult clears its internal error flag. I/O-checking must be off
{$I-}.

 Sample Code :



 {IOResult.PAS}

 {Sample code for the IOResult function.}

 { For Windows: }
 { uses WinCrt; }

 var F: file of Byte;
 begin
  { Get file name command line }
  Assign(F, ParamStr(1));
  {$I-}
  Reset(F);
  {$I+}
  if IOResult = 0 then
    Writeln('File size in bytes: ', FileSize(F))
  else
    Writeln('File not found');
 end.
