 +- Expression -------------------------+
 |                                      |
 +--------------------------------------+

The Expression input box shows the word at the
cursor in the Edit window, which is the
default expression.

To evaluate the default expression, press
Enter. You can edit or replace it first, using
the history list to enter previously entered
expressions.

To extend the default expression (by copying
additional characters from the Edit window),
press the Right arrow.

If the debugger can evaluate the expression,
it displays the value in the Result input box.

If the expression refers to a variable or
simple data element, you can move the cursor
to the New Value input box and enter an
expression as the new value.

To close the dialog box, press Esc. If you've
changed the contents of the New Value input
box and have not pressed Enter, the debugger
will ignore any changes you've made when you
close the dialog box.
