 -c option
 ----------
The optional -c switch can also be specified as /c. It can appear in any
argument position on the command line.

When you use this option, any existing command table in the configuration
file is discarded before TEMC processes the script file. If you do not use
this option, the key bindings in the script file are merged with those
already defined in the configuration file.
