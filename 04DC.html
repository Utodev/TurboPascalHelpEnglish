 SetDate (procedure)     (Dos unit and WinDos unit)
 ---------------------------------------------------
Sets the current date in the operating system.

 Declaration:
 procedure SetDate(Year, Month, Day: Word);

 Target:
Windows, Real, Protected

 Remarks:
Valid parameter ranges are Year 1980..2099, Month 1..12, and Day 1..31. If
the date is invalid, the request is ignored.

 See Also:
 GetDate
 GetTime
 SetTime

 Sample Code:



 {SetDate.PAS}

 {Sample code for the SetDate procedure.}

 { For Windows: }
 { uses WinDos; }

 uses Dos;

 begin
   { Set the system date to 2/10/89 }
   SetDate(1989,2,10);
 end.
