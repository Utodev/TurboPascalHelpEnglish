 Variables of Type Registers     (Dos unit)
 -------------------------------------------
The Intr and MsDos procedures use variables of type Registers to specify
the input register contents and examine the output register contents of a
software interrupt.

 type
     (Registers record used by Intr and MsDos)
   Registers = record
     case Integer of
     0: (AX, BX, CX, DX, BP, SI, DI, DS, ES, Flags: Word);
     1: (AL, AH, BL, BH, CL, CH, DL, DH: Byte);
   end;
