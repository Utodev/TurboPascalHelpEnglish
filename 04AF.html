 OvrInit (procedure)     (Overlay unit)
 ---------------------------------------
Initializes overlay manager, opens overlay file.

 Declaration:
 procedure OvrInit(FileName:string );

 Target:
Real

 Remarks:
Errors are reported in the OvrResult variable.

OvrInit must be called before any of the other overlay manager procedures.

 See Also:
 OvrGetBuf
 OvrInitEMS
 OvrSetBuf

 Sample Code:



 {Ovrinit.PAS}

 {Sample code for the OvrInit procedure: Runs only in DOS real mode}

 uses Overlay;

 begin
  OvrInit('EDITOR.OVR');
  if OvrResult <> ovrOk then
  begin
    case OvrResult of
      ovrError: Writeln('Program has no overlays.');
      ovrNotFound: Writeln('Overlay file not found.');
    end;
    Halt(1);
  end;
 end.
