 Overlay Typed Constants in SYSTEM.TPU
 --------------------------------------
The overlay manager uses the following typed constants:

  OvrCodeList:  Word    = 0;   { Code segment list }
  OvrDebugPtr:  pointer = nil; { Debug hook }
  OvrDosHandle: Word    = 0;   { Overlay file handle }
  OvrEMSHandle: Word    = 0;   { Overlay EMS handle }
  OvrHeapEnd:   Word    = 0;   { Overlay buf end }
  OvrHeapOrg:   Word    = 0;   { Overlay buf origin }
  OvrHeapPtr:   Word    = 0;   { Overlay buf pointer }
  OvrHeapSize:  Word    = 0;   { Initial buf size }
  OvrLoadList:  Word    = 0;   { Loaded overlay list }
