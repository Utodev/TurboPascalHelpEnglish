The Swap File Directory input box is where you
specify the drive and path of a "fast" swap
area, such as a RAM disk.

You use a fast swap area if no EMS is
available on your system.

If no swap directory is specified, the IDE
creates a default swap file in the current
directory.

 +------------------------------------+
 | Default setting: Current directory |
 | Command-line equivalent: /S        |
