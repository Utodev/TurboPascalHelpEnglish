 | Undefined external |
 +--------------------+

The EXTERN procedure or function doesn't
have a matching PUBLIC definition in an
object file.

Make sure you have specified all object files
in {$L filename} directives, and check the
spelling of the procedure or function
identifier in the .ASM file.
