 - Window     (Alt+W)
 ---------
The Window menu contains commands for
manipulating and opening windows.

Most of the windows you open from this menu
have all the standard window elements, like
scroll bars, a close box, and zoom icons.

For more about windows and their elements,
see the Window Basics Help screen.
