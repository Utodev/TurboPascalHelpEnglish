 - Help|Files...
 ------------------
The Help|Files... Option provides access to
the Install Help Files dialog box.

This option allows you to install or remove
additional Help files. Once you have installed additional
Help files, the index will be rebuilt. This may take a few
moments the first time but the merged index will be available
until you close the IDE.

The TURBO.TPH file will always be loaded by default.
