 UpCase (function)
 ------------------
Converts a character to uppercase.

 Declaration:
 function UpCase(Ch: Char): Char;

 Target:
Windows, Real, Protected

 Remarks:
Ch is an expression of type Char. The result of type Char is Ch converted
to uppercase. Character values not in the range a..z are unaffected.

 Sample Code:



 {UpCase.PAS}

 {Sample code for the UpCase function.}

 { For Windows: }
 { uses WinCrt; }

 var
   s : string;
   i : Integer;
 begin
   Write('Enter a string: ');
   Readln(s);
   for i := 1 to Length(s) do
     s[i] := UpCase(s[i]);
   Writeln('Here it is in all uppercase: ',s);
 end.
