 +-------------------------------------+
 | [X] Load.TPL  file (or LOAD.TPL)    |
 +-------------------------------------+

[X] When you check Load.TPL (and click
OK in the Startup Options dialog box), the
IDE loads the file TURBO.TPL the next time
you start TURBO.EXE or TPX.EXE or LOAD.TPL the
next time you start BP.EXE.

For BP, this option loads the appropriate run-time library depending
on the specified target.

  Real             Windows            Protected Mode
 ---------------------------------------------------
 TURBO.TPL         TPW.TPL              TPP.TPL

[ ] If the option is not checked, run-time library
is not loaded at startup, and the System unit
file must be available when you
attempt to compile or debug your programs.

To increase the DOS IDE's capacity, turn off this
option and use the TPUMOVER to extract
SYSTEM.TPU from the run-time library.

 +-----------------------------------+
 | Default setting: On ([X] or /T+)  |
 | Command-line equivalent: /T       |
 +-----------------------------------+
