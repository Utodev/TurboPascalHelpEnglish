 +----------------------+
 | Sub-browsing         |
 |  ( ) New browser     |
 |  (.) Replace current |
 +----------------------+

When you select a new subgroup to browse, the current browser window
will remain open by default and the information in the window is replaced
each time you browse. Change this option if you want to open a new browser
window each time you browse.
