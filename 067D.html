 Interrupt-Vector Constants in SYSTEM.TPU
 -----------------------------------------
The following variables, all of type pointer, save the original values of
interrupt vectors that the compiler changes.

In SaveIntXX, XX is the interrupt number in hex:

  SaveInt00, SaveInt02, SaveInt1B, SaveInt21,
  SaveInt23, SaveInt24, SaveInt34, SaveInt35,
  SaveInt36, SaveInt37, SaveInt38, SaveInt39,
  SaveInt3A, SaveInt3B, SaveInt3C, SaveInt3D,
  SaveInt3E, SaveInt3F, SaveInt75: pointer;
