   /T Command-Line Option
 ------------------------
The /T command-line option tells the compiler where to find the:
  - configuration file
  - resident library file

If these files are in the current directory or in the directory with the
.EXE file, you do not need to use this option.

If you use this option, it must be the first command-line argument.
