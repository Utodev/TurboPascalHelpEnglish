 ScrollTo (procedure)     (WinCrt unit)
 ---------------------------------------
Scrolls the CRT window to show the virtual screen location given by (X,Y)
in the upper left corner.

 Declaration:
 procedure ScrollTo(X, Y: Integer);

 Target:
Windows

 Remarks:
The upper left corner of the virtual screen corresponds to (0,0).

 Sample Code:



 {ScrollTo.PAS}

 {Sample code for the ScrollTo procedure.}

 uses WinCrt;

 begin
  GotoXY(1,10);
  Writeln('Hello');
  Writeln('Type in a line and press Enter.');
  Readln;
  ScrollTo(0,10);
 end.
