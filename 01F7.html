 +----------------+
 | Path not found |
 +----------------+

The routines Reset, Rewrite, Append, Rename,
and Erase report this error if the name
assigned to the file variable is invalid or
specifies a non-existent subdirectory.

The routines ChDir, MkDir, or RmDir report
this error if the path is invalid or if it
specifies a non-existent subdirectory.
