 FileSearch (function)     (WinDos unit)
 ----------------------------------------
Searches for a file.

 Declaration:
 function FileSearch(Dest, Name, List: PChar): PChar;

 Target:
Windows

 Remarks:
Searches in a list of directories (DirList). The directories in DirList must
be separated by semicolons.

 See Also:
 FileExpand
 FileSplit

 Sample Code:



 {FileSch.PAS}

 {Sample code for the FileSearch function.}

 { For Windows: }
 { uses WinCrt, WinDos; }

 uses WinDos;

 var
  S: array[0..fsPathName] of Char;
 begin
  FileSearch(S, 'TURBO.EXE', GetEnvVar('PATH'));
  if S[0] = #0 then
    Writeln('TURBO.EXE not found')
  else
    Writeln('Found as ', FileExpand(S, S));
 end.
