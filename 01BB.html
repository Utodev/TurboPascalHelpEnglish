 - /D Command-line Option
 -------------------------
/D places  Turbo  Pascal in dual-monitor mode
if it detects appropriate hardware.

Default setting: Off (/D-)

 See Also:
  Other command-line options  
  O|E|Startup [X] Dual Monitor Support
