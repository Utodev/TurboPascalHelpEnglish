 +---------------------+
 | [ ] Smart callbacks | Windows Target
 +---------------------+

This option enables or disables the generation of Smart callback
information. It is used when compiling programs targeted for Windows.

This option is ignored if the target is Real Mode or Protected.

[ ] When Smart Callbacks is not checked, there is no need to
use MakeProcInstance and FreeProcInstance.

[X] When you check Smart Callbacks, an application is required
to use the MakeProcInstance and FreeProcInstance Windows API
routines when creating callback routines.

Smart Callbacks is equivalent to {$K}.
