 ParamCount (function)
 ----------------------
Returns the number of parameters passed to the program on the command line.

 Declaration:
 function ParamCount: Word;

 Target:
Windows, Real, Protected

 Remarks:
Blanks and tabs serve as separators.

 See Also:
 ParamStr

 Sample Code:



 {Parcount.PAS}

 {Sample code for the ParamCount function.}

 begin
  if ParamCount = 0 then
    Writeln('No parameters on command line')
  else
    Writeln(ParamCount, ' parameter(s)');
 end.
