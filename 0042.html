 - Run|Program Reset  (Ctrl+F2)
 -------------------------------
The Program Reset command:

 » stops the current debugging session
 » releases memory your program has allocated
 » closes any open files that your program was
   using

This command releases some of the memory used
for debugging information, so you should use
it at the end of a debugging session if you're
going to work on a program that requires as
much memory as possible to compile.
