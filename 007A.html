 -------------------------
 +--------+
 |Shift-F1|  Turbo Pascal's On-line Help comes
 +--------+  with a comprehensive alphabetic
             index.

Once you're in Help, there are 3 ways to bring up the Help Index:

 1) Press Shift-F1.
 2) If you're in a Turbo Help (dialog-style)
    window, choose the Index button.
 3) If you're in a regular Help window (or if
    you don't have any Help windows open),
    choose Index from the Help or local Help menu.

You can scroll through the index, move the
cursor to the item you want, or type the first
few letters of the item to activate Turbo
Pascal's incremental search feature.

Once the item you want is highlighted, press
Enter (or double-click) to go to the Help
screen for that item.
