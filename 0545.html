 DiskFree (function)     (Dos unit and WinDos unit)
 ---------------------------------------------------
Returns the number of free bytes on a specified disk drive.

 Declaration:
 function DiskFree(Drive: Byte): Longint;

 Target:
Windows, Real, Protected

 Remarks:
where
   0     Default drive
   1     Drive A
   2     Drive B
   3     Drive C
etc.

 Return Value:
-1 if the drive number is invalid.

 See Also:
 DiskSize
 GetDir

 Sample Code:



 {Diskfree.PAS}

 {Sample code for the DiskFree function.}

 { For Windows: }
 { uses WinDos, WinCrt; }

 uses Dos;

 begin
  Writeln(DiskFree(0) div 1024, ' Kbytes free ');
 end.
