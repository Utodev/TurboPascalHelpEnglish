 Parameter Directives
 ----------------------
Parameter directives specify parameters that affect how a program is
compiled.

   Parameter Directive   | Meaning
   =========================================================================
   $C Attribute          | Code Segment Attribute (Windows and Protected
                         | mode)
   $D Text               | Description (Windows and Protected mode)
   $I FileName           | Include File
   $G UnitName           | Group Unit Segments (Windows and Protected mode)
   $L FileName           | Link Object File
   $M StackSize,HeapSize | Memory Allocation Sizes
   $O UnitName           | Overlay Unit Name (Real mode only)
   $R FileName           | Resource File (Windows and Protected mode)
   $S SegSize            | Segment Size Preference (Windows and Protected
                         | mode)

They must have at least one blank space between the directive name and its
parameters. For example:

 {$I TYPES.INC}
 {$O MOONUNIT}
