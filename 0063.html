 - Options|Save As...
 ---------------------
Options|Save As leads to the Save Options
dialog box, where Turbo Pascal saves settings
you have made in the Search dialog boxes
(both Search|Find and Search|Replace), in the
Compile|Primary File commands, and all the
settings under the Options menu.

All options and the editor command table are
stored in the configuration file; history lists, the
desktop state, and breakpoint locations are
stored in TURBO.DSK, TPX.DSK or BP.DSK, the default desktop file.

When you start Turbo Pascal, it looks in the
current directory for the saved option file.
If it doesn't find the file there, it looks
for it in the Turbo Pascal directory (where
TURBO.EXE, TPX.EXE or BP.EXE is run from).
