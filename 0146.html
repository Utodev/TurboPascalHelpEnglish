 - Source Tracking
 ------------------
The Source Tracking options affects the ObjectBrowser Reference view,
the Messages window, and the integrated debugger.

If the line referenced isn't open in an edit window, the IDE
opens it and highlights the appropriate line.

 +--------------------+
 | ( ) New window     |
 +--------------------+
If you select the New Window option, the newly opened file appears in a
new edit window.

 +--------------------+
 | ( ) Current window |
 +--------------------+
If you select the Current Window, the newly opened file replaces the file
in the current edit window.
