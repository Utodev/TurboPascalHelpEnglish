 - Help     (Alt+H)
 -------
The Help menu provides access to on-line Help,
which comes up in a special Help window. The
text in the Help window is a help screen.

The Help system provides information on
virtually all aspects of the integrated
environment and Turbo Pascal.

(Also, don't forget that one-line menu and
dialog-box hints appear on the status line
whenever you select a menu command or dialog
box item.)
