 StrLen (function)     (Strings unit)
 -------------------------------------
Returns the number of characters in Str.

 Declaration:
 function StrLen(Str: PChar): Word;

 Target:
Windows, Real, Protected

 Remarks:
StrLen returns the number of characters in Str, not counting the null
terminator.

 See Also:
 StrEnd

 Sample Code:



 {StrLen.PAS}

 {Sample code for the StrLen function.}

 { For Windows: }
 { uses Strings, WinCrt; }

 uses Strings;

 var
  S: array[0..79] of Char;
 begin
  Readln(S);
  Writeln('String length is ', StrLen(S));
 end.
