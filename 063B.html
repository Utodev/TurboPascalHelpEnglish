  $CAP EDIT
 ------------
This macro tells the IDE to redirect program output into a standard file.

After the transfer program is completed, a new edit window is created, and
the captured output is displayed.

The captured output resides in a special edit window titled Transfer
Output.

For $CAP EDIT to work correctly, the transfer program must write to DOS
standard output.
