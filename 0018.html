 - Edit  (Alt-E)
 ----------------
The Edit menu provides commands to cut, copy,
and paste text in Edit windows.

You can also open a Clipboard window to view
or edit its contents.

You can choose from these Edit menu commands:

 Undo              
 Redo              
 Cut               
 Copy              
 Paste             
 Clear             
 Show Clipboard
