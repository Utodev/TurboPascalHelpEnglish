 - Search|Find Procedure...
 ---------------------------
This command displays the Find Procedure
dialog box, where you enter the name of a
procedure or function to search for.

This command is available after you have compiled
the program in the active edit window.

Enter the name of a procedure or press Down
arrow to choose a name from the history list.

Find Procedure is different than the Find
command; it finds the implementation of the
procedure or function, not instances of the procedure's
or function's use.
