  Software floating point
 -------------------------
In the {$N-} state (on by default), the code generated performs all
real-type calculations in software by calling run-time library routines.

Because of speed and code size, only operations on variables of type real
are allowed in this state.

Attempt to compile statements that operate on Single, Double, Extended, and
Comp types generate an error.
