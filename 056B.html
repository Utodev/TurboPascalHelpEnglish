 GetArgStr (function)     (WinDos unit)
 ---------------------------------------
Returns the command-line parameter specified by Index.

 Declaration:
 function GetArgStr(Dest: PChar; Index: Integer; MaxLen: Word): PChar;

 Target:
Windows

 Return Value:
 Parameter                  | Return Value
 =========================================================================
 Index < 0 or > GetArGCount | empty string
 Index = 0                  | filename of current module; Dest = returned
                            | value

 See Also:
 GetArgCount

 Sample Code:



 {GetArgSt.PAS}

 {Sample code for the GetArgStr function.}

 { For Windows: }
 { uses WinDos, WinCrt; }

 uses WinDos;

 var
  ArgN: PChar;

 begin
  GetMem(ArgN, 20);
  GetArgStr(ArgN, 1, 19);
  Writeln('The 1st argument to the prompt (in addition to the');
  Writeln('', ArgN, '.');
 end.
