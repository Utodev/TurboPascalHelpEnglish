 +- Resource directories ---------------+
 |                                      |
 +--------------------------------------+
This input box specifies the directories
that contain the resource files for your
Windows application. (BP and BPW only)

When the compiler encounters a {$R filename} directive,
it looks first in the current
directory, then in the directories specified
in this Resource Directories input box.

 See also:
 Guidelines for entering directory names
