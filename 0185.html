You use Foreground and Background palettes
on black-and-white and color systems to
modify the foreground and background colors
of the current IDE item (view).

To change the color for the selected item in
the current group, use the cursor keys or
mouse to choose a new color from the palette.

The block of display text below the palettes
shows the current color settings.
