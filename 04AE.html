 OvrGetRetry (function)
 -----------------------
Returns the current size of the overlay buffer probation area.

 Declaration:
 function OvrGetRetry : Longint;

 Target:
Real

 See Also:
 OvrSetRetry

 Sample Code:



  {OvrGRtry.PAS}

 { Example for OvrGetRetry and OvrSetRetry: Runs only in DOS real mode }

 {$M 16384,65536,655360 }
 uses Overlay;
 const
   ExtraSize = 49152; { 48K}
 begin
     OvrInit('MYOVR.OVR');
     WriteLn('Initial size of overlay ',
             'buffer is ', OvrGetBuf,
             ' bytes.');
     OvrSetBuf(OvrGetBuf+ExtraSize);
     WriteLn('Overlay buffer now increased ',
             'to ', OvrGetBuf,' bytes.');
     OvrSetRetry(OvrGetBuf+ExtraSize div 3);
     Writeln('The current size of the probationary area is ',
     OvrGetRetry);
 end.
