 MemAlloc (function)     (OMemory unit)
 ---------------------------------------
Allocates memory on the heap and returns a pointer to the block.

 Declaration:
 function MemAlloc(Size: Word): Pointer;

 Target:
Windows

 Remarks:
Allocates Size bytes of memory on the heap and returns a pointer to the
block. If a block of the requested size cannot be allocated, a value of nil
is returned.

MemAlloc will not allow the allocation acess the safety pool. A block
allocated by MemAlloc can be disposed using the FreeMem standard procedure.
