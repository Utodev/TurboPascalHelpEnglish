 -------------------------------
  Transfer Arguments dialog box
 -------------------------------
The Transfer Arguments dialog box comes up
when you choose a Transfer item that contains
a $PROMPT transfer macro in its command line.

You can change or add to the parameter string
before executing the Transfer item.

The position of the $PROMPT macro in the
command line determines what you see in the
Arguments input box.

Anything preceding the $PROMPT macro is
treated as a constant and therefore won't be
visible.

If you want to change a constant item, choose
the Edit button in the Options|Transfer
dialog box.
