 - Help on the Program Parameters dialog box
 -------------------------------------------
The Program Parameters dialog box consists of
an input box and the standard OK, Cancel, and
Help buttons.

 +- Parameters -------------------------+
 |                                      |
 +--------------------------------------+
The Parameters input box is where you enter
the parameters that you want Turbo Pascal to
pass to your running program.

You don't need to enter the program name in
this input box.

Parameters take effect only when your program
is started.
