 Trunc (function)
 -----------------
Truncates a real-type value to an Integer-type value.

 Declaration:
 function Trunc(X: Real): Longint;

 Target:
Windows, Real, Protected

 Remarks:
X is a real-type expression. Trunc returns a Longint value that is the
value of X rounded toward zero.

 Restrictions:
A run-time error occurs if the truncated value of X is not within the
Longint range.

 See Also:
 Int
 Round

 Sample Code:



 {Trunc.PAS}

 {Sample code for the Trunc function.}

 { For Windows: }
 { uses WinCrt; }

 begin
   Writeln(14, '  becomes ', Trunc(14));
   Writeln(15, ' becomes ', Trunc(15));
   Writeln(-14, ' becomes ', Trunc(-14));
   Writeln(-15, ' becomes ', Trunc(-15));
 end.
