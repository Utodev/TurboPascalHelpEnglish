 Bar (procedure)     (Graph unit)
 ---------------------------------
Draws a bar using the current fill style and fill color.

 Declaration:
 procedure Bar(x1, y1, x2, y2: Integer);

 Target:
Real, Protected

 Remarks:
Bar draws a filled-in rectangle (used in bar charts, for example). Uses the
pattern and color defined by SetFillStyle or SetFillPattern. To draw an
outlined bar, call Bar3D with a depth of zero.

 Restrictions:
Must be in graphics mode.

 See Also:
 Bar3D
 SetFillPattern
 SetFillStyle
 SetLineStyle


 Sample Code:



 {Bar.PAS}

 {Sample code for the Bar procedure.}

 uses
  Graph;

 var
  Gd, Gm, I, Width: Integer;
 begin
  Gd := Detect; InitGraph(Gd, Gm,' ');
  if GraphResult <> grOk then Halt(1);
  Width := 10;
  for I := 1 to 5 do
    Bar(I*Width, I*10, Succ(I)*Width, 200);
  ReadLn;
  CloseGraph;
 end.
