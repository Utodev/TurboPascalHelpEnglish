 | Contents (Help local menu)
 ----------------------------
The Help Contents screen provides a summary of several
Help topics you can choose from.

There are three ways to get to the Help Contents
screen:
 1) If you're in a Turbo Help (dialog-style)
    window, choose the Contents button.

 »  If you're in a regular Help window (or if
    you don't have any Help windows open):
 2) choose Contents from the Help menu.
 3) right-click your mouse to display the local
    Help menu and choose Contents.
