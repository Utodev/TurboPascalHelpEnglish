TURBO only

The Window Heap Size input box is where you
change the size of the window heap.

If you don't need a lot of windows open on
your desktop, reduce the window heap size to
make more memory available for your programs.

The default provides for good capacity and
ample window space.

 +-----------------------------+
 | Default size: 32K           |
 | Minimum size: 24K           |
 | Maximum size: 64K           |
 | Command-line equivalent: /W |
 +-----------------------------+
