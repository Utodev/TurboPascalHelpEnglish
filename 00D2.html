 - Help on the Find Error dialog box
 ------------------------------------
The Find Error dialog box consists of an input
box with a history list, and the standard OK,
Cancel, and Help buttons.

When a run-time error occurs, the program
gives the error's memory location (segment
and offset).

To locate the error again, you enter the
given segment:offset address in the Find
Error input box.
