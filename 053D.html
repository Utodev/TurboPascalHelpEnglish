 CreateDir (procedure)     (WinDos unit)
 ----------------------------------------
Creates a new subdirectory.

 Declaration:
 procedure CreateDir(Dir: PChar)

 Target:
Windows

 Remarks:
Performs the same functions as MkDir, but uses a null-terminated string
rather than a Pascal-style string.

 See Also:
 GetCurDir
 SetCurDir
 RemoveDir

 Sample Code:



 {CreateDr.PAS}

 {Sample code for the CreateDir procedure.}

 { For Windows: }
 { uses WinDos, WinCrt; }

 uses WinDos;

 const
  Create: PChar = 'C:\NEWDIR';
  CDrive: Byte = 3;

 begin
  CreateDir(Create);
  Writeln('There is now a new directory at ', Create,'.');
 end.
