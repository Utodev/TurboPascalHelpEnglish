 Printer Setup Dialog Box
 ------------------------

The Printer Setup dialog box has two input boxes, one checkbox,
and OK, Cancel and Help buttons.

 + Filter Path ------------------------+
 |                                     |
 +-------------------------------------+

The Filter Path input box allows you to specify a path name for the printer
filter file. The default setting is PRNFLTR.

 + Command Line -----------------------+
 |                                     |
 +-------------------------------------+

The Command Line input box allows you to specify command line parameters
for your printer. The default setting is $NOSWAP /EPSON.

 [X]  Send highlighting escape codes

The Send Highlighting Escape codes checkbox specifies whether the
 Turbo  Pascal IDE includes escape code sequences to print
highlighting information as bold, italic, and underline.
