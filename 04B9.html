 Pred (function)
 ----------------
Returns the predecessor of the argument.

 Declaration:
 function Pred(X): < Same type as parameter >;

 Target:
Windows, Real, Protected

 Remarks:
X is an ordinal-type expression. The result, of the same type as X, is the
predecessor of X.

 See Also:
 Dec
 Inc
 Succ

 Sample Code:



 {Pred.PAS}

 {Sample code for the Pred and Succ functions.}

 { For Windows: }
 { uses WinCrt; }

 type
   Colors = (RED,BLUE,GREEN);
 begin
   Writeln('The predecessor of 5 is ',
           Pred(5));
   Writeln('The successor of 10 is ',
           Succ(10));
   if Succ(RED) = BLUE then
     Writeln('In the type Colors, RED is ',
             'the predecessor of BLUE.');
 end.
