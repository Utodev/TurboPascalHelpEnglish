 +-----------------------------------------------+
 | Options                                       |
 |  [ ] Auto track source                        |
 +-----------------------------------------------+

When this option is selected, the source code line in the editor will be
highlighted as the symbol is referenced in the ObjectBrowser or Messages
window.

If this option is not selected, the source code line will not be
highlighted.

This option is used for both the Messages window and the ObjectBrowser.
