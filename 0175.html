 +--------------------------+
 | [X] Graphics Screen Save |
 +--------------------------+

[X] When you check Graphics Screen Save (and
click OK in the Startup Options dialog box),
then start up the IDE again, it performs a
full graphics memory save when running your
graphics-mode programs.

With this option on, the IDE reserves another
8K for the graphics buffer (which will be
placed in EMS, if it's available).

Check Graphics Screen Save while you debug
graphics programs on EGA, VGA, and MCGA
systems.

 +-----------------------------------+
 | Default setting: Off ([ ] or /G-) |
 | Command-line equivalent: /G       |
 +-----------------------------------+
