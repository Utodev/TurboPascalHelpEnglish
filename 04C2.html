 Readln (procedure)
 -------------------
Executes the Read procedure then skips to the next line of the file.

 Declaration:
 procedure Readln([ var F: Text; ] V1 [, V2, ...,Vn ]);

 Target:
Windows, Real, Protected

 Remarks:
After executing Read, Readln skips to the beginning of the next line of the
file.

 See Also:
 Read

 Sample Code:



 {Readln.PAS}

 {Sample code for the Readln and Writeln procedures.}

 { For Windows: }
 { uses WinCrt; }

 var
   s : String;
 begin
   Write('Enter a line of text: ');
   Readln(s);
   Writeln('You typed: ',s);
   Writeln('Hit  to exit');
   Readln;
 end.
