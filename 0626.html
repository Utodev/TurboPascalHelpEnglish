 -u option
 ----------
The optional -u switch can be specified as /u. It can appear in any
argument position on the command line.

You use the -u option to modify the CUA command set. (By default, TEMC
modifies the commands used by the IDE when you choose the Alternate command
set in the Preferences dialog box.)

This option is not used by Borland Pascal.
