 - Help on the Target dialog box
 --------------------------------
The Target dialog box consists of one group of
check boxes and the standard OK, Cancel, and
Help buttons.

 +---------------------------------+
 | Target                          |
 |  [X] Real mode Application      |
 |  [ ] Protected mode Application |
 |  [ ] Windows Application        |
 +---------------------------------+

The Target option tells the compiler what type of application
to generate.

Set the compiler options for each target in Options|Compiler Options.
