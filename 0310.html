 Changing the way the ObjectBrowser displays information
 --------------------------------------------------------
You can control how the ObjectBrowser displays information with the
Browser Options dialog box.

 » Select which symbols the ObjectBrowser displays
 » Select to see scope or reference information as the default view
   when you bring up the ObjectBrowser
 » Select to open a new browser window or replace the current
   browser window when you sub-browse
 » Select which view you want displayed as the default: Scope or
   Reference
