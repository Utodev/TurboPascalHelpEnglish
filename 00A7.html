The Files list box lists the names of files in
the current directory that match the file-name
mask in the Name input box, plus the parent
directory and all subdirectories.

To open a file, click the file name (or
move the cursor to it), then choose Open or
Replace.

If you're using the keyboard, press Enter to
open the selected file.
