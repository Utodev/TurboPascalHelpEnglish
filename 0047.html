 - Compile|Compile    (Alt+F9)
 ------------------------------
The Compile|Compile command compiles the file
in the active Edit window.

When Turbo Pascal is compiling or making, a
status box pops up to display the compilation
progress and results.

When compiling (or making) is complete, press
any key to remove the status box.

If an error occurred, a message describing it
appears at the top of the Edit window.
