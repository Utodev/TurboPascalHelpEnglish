 | Debug|Evaluate/Modify
 | Evaluate/Modify (Local menu)
 ------------------------------
With the Evaluate/Modify command, you can

 » Evaluate a variable or expression.
 » View the value of any variable or other
   data item.
 » Alter the value of simple data items.

While your program is running, Evaluate/Modify
opens the Evaluate and Modify dialog box,
which contains three input boxes: Evaluate,
Result, and New Value.
