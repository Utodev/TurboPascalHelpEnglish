 +- EXE and TPU directory --------------+
 |                                      |
 +--------------------------------------+
This input box specifies the directory that
stores your .EXE and unit (object code) files.

If this entry is blank, your .EXE and unit
files are stored in the directory where your
source code is.

You cannot specify multiple EXE and TPU directories.

If the Map File setting in the Linker dialog
box is set to anything other than Off, .MAP
files are also stored in this directory.

See also:

 Guidelines for entering directory names
