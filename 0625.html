 Calling TEMC
 -------------
TEMC is invoked from the DOS command line. Type:

 temc [-c] [-u-] <script file> <config file>

where:

 » -c discards any existing command table in your configuration file
 » -u causes TEMC to modify the CUA command set
 » script file is the TEML script file. The default extension is .TEM.
 » config file is the current configuration file.
   The default extension is .TC for C and C++ and .CMD for Borland Pascal
   for Windows or .TP for Borland Pascal for DOS.

The [-u-] option is not used by Borland Pascal.
