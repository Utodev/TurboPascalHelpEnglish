 | [X] Standalone  |
 +-----------------+

[X] Programs that are linked with Standalone
checked (on) can only be debugged with Turbo
Debugger.

You can still run these programs in Turbo
Pascal, however.

The Standalone option produces complete
debugging information for a module only
if that module has been compiled with the
Options|Compiler Debug Information option checked.

Standalone produces local symbol information
for a module only if that module has been
compiled with the O|C Local Symbols option
checked.
