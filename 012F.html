 ------------------
  Tools dialog box
 ------------------
The Tools dialog box is where you can
  » review the programs you've already set up
    as transfer programs
  » add new transfer programs
  » edit or delete existing ones

 +- Program Titles -------+
 |   Grep                »|
 |   Turbo Assembler     ±|
 |   Turbo Debugger      ±|
 |   Turbo Profiler      ±|

The Program Titles list box lists transfer
programs that have been installed and are
ready to run.

 +----------+
 | [ Edit ] |
 +----------+
Edit changes a program in the Program
Titles list box and the Tools menu.

 +---------+
 | [ New ] |
 +---------+
New adds a program in the Program
Titles list box and the Tools menu.

 +------------+
 | [ Delete ] |
 +------------+
Delete removes the currently selected program
from the list and from the Tools menu.
