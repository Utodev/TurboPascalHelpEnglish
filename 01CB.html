 - Mem, MemL, and MemW
 ----------------------
 Turbo  Pascal implements three predefined
arrays to directly access memory: Mem, MemL,
and MemW.

 » Each component of Mem is a Byte
 » Each component of MemW is a Word
 » Each component of MemL is a Longint.

The Mem arrays use a special syntax for
indexes: two expressions of the Integer-type
Word, separated by a colon, specify the
segment base and offset of the memory
location to access.
