 TrackCursor (procedure)     (WinCrt unit)
 ------------------------------------------
Scrolls the CRT window if necessary to ensure that the cursor is visible.

 Target:
Windows

 Declaration:
 procedure TrackCursor;

 See Also:
 ScrollTo

 Sample Code:



  {TrackCur.PAS}

 {Sample code for the TrackCursor procedure.}

 uses WinCrt;

 var
   x: integer;

 begin
   for x := 1 to 30 do
     Write('Xx');
   TrackCursor;
   Readln;
 end.
