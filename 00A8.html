In the Load a File dialog box, if you choose
the Open button instead of the Replace button,
 Turbo  Pascal opens a new Edit window and
places the selected file in that window.

If you choose Replace, the selected file
replaces the file that's in the active Edit
window instead of opening in a new window.

Once you've typed in or selected the file you
want, choose the Open button (choose Cancel if
you change your mind).

You can also just press Enter once the file is
selected, or you can double-click the file
name.
