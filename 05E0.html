 assignment operator ( := )
 ----------------------------
Assigns the value of an expression to a variable.

 Syntax:
 variable := expression

 Remarks:
The variable must be assignment-compatible with the result type of the
expression.

 Example:
 X    := Y;
 Done := (I > 0) and (I < 100)
 A[I] := A[I] + 1;
