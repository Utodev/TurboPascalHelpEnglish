 SelectorInc (variable)     (System unit)
 -----------------------------------------
Selector increment value.

 Target:
Protected; Real; Windows

 Declaration:
  var SelectorInc: Word;

 Remarks:
SelectorInc contains the value that must be added to or subtracted from the
selector part of a pointer to increment or decrement the pointer by 64K
bytes. In real mode, SelectorInc always contains $1000, but in protected
mode the actual value can vary.
