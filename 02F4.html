The Title input box is where you type
in a short description of the program you're
adding.

Use the Backspace or Del keys to correct any
errors.

If you want your program to have a keyboard
shortcut (like the "S" in the Save command or
the "t" in the Cut command), include a tilde
(~) in the name, just before and after the shortcut
character (like this: ~S~ave or Cu~t~).

The character that is surrounded by tildes will
appear in bold or in a special color in the
Tools menu.

You can press that shortcut key to choose the
transfer program from the Tools menu.
