02BF
 
 +---------------------+
 | Heap overflow error |
 +---------------------+

New and GetMem report this error when there
is not enough free space in the heap to
allocate a block of the requested size.

If you get Heap Overflow errors when running
your program from within the IDE, try exiting
 Turbo  Pascal (File|Exit) and running your
program from the DOS command line.

See run-time error #1: Out of Memory for
suggestions on how to regain memory.

For a complete discussion of the heap
manager, refer to the chapter "Memory Issues"
in the  Turbo  Pascal Programmer's Guide.
