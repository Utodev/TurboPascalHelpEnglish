 +- Watch expression ----------------------+
 |                                         |
 +-----------------------------------------+

The Watch Expression input box is where you
edit the default watch expression.

You can use the history list to reenter
previously entered watch expressions.

When you edit the expression and press Enter
or click OK, the debugger replaces the
original version of the expression with the
edited one.

To edit a watch expression from inside the
Watches window, select the expression and
press Enter.
