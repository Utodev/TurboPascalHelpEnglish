 StrLCopy (function)     (Strings unit)
 ---------------------------------------
Copies characters from one string to another.

 Declaration:
 function StrLCopy(Dest, Source: PChar; MaxLen: Word): PChar;

 Target:
Windows, Real, Protected

 Remarks:
StrLCopy copies at most MaxLen characters from Source to Dest and returns
Dest. The SizeOf standard function can be used to determine the MaxLen
parameter.

 See Also:
 StrCopy

 Sample Code:



 {StrLCopy.PAS}

 {Sample code for the StrLCopy function.}

 { For Windows: }
 { uses Strings, WinCrt; }

 uses Strings;

 var
  S: array[0..9] of Char;
 begin
  StrLCopy(S, 'Turbo Pascal', SizeOf(S) - 1);
  Writeln(S);
 end.
