 - Setting Breakpoints
 ----------------------
Before you compile a source file, you can set
a breakpoint on any line, even a blank line
or a comment.

When you compile and run the file, Turbo
Pascal validates any breakpoints that are set
and gives you a chance to remove, ignore, or
change invalid breakpoints.

When you are debugging the file, Turbo Pascal
knows which lines contain executable
statements, and will warn you if you try to
set invalid breakpoints.
