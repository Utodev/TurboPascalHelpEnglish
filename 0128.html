 +- Object directories -----------------+
 |                                      |
 +--------------------------------------+
This input box specifies the directories
that contain your .OBJ files (assembly
language routines).

When Turbo Pascal encounters a {$L filename}
directive, it looks first in the current
directory, then in the directories specified
in this Object Directories input box.

See also:

 Guidelines for entering directory names
