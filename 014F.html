 +-----------------------+
 | [X] Use Tab Character |
 +-----------------------+

[X] When you check Use Tab Character, Turbo
Pascal inserts a true tab character (ASCII 9)
when you press Tab.

[ ] When this option is not checked, Turbo
Pascal replaces tabs with spaces.

Tab Size in the Editor Options dialog box
determines the number of spaces used to
replace a Tab.
