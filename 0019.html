 - Search     (Alt+S)
 ---------
The Search menu provides commands to search
for text, procedure declarations, and error
locations in your files.

These are the available options:

 Find                          
 Replace                       
 Search Again                  
 Go to Line Number             
 Show last compiler error      
 Find Error                    
 Find Procedure                
 Previous Browser (BP and BPW) 
 Objects (BP and BPW)          
 Units (BP and BPW)            
 Globals (BP and BPW)          
 Symbol (BP and BPW)
