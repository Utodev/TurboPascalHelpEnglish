 GetCurDir (function)     (WinDos unit)
 ---------------------------------------
 Returns the current directory of a specified drive.

 Declaration:
 function GetCurDir(Dir: PChar; Drive: Byte): PChar

 Target:
Windows

 Remarks:
where
   0      Default drive
   1      Drive A
   2      Drive B
   3      Drive C
 etc.

 See Also:
 SetCurDir
 CreateDir
 RemoveDir

 Sample Code:



 {GetCurDr.PAS}

 {Sample code for the GetCurDir function.}

 { For Windows: }
 { uses WinDos, WinCrt; }

 uses WinDos;

 const
  CDrive:Byte = 3;

 var
  CurDir: PChar;

 begin
  GetMem(CurDir, 80);
  GetCurDir(CurDir, CDrive);
  Writeln('The current directory on drive c is ', CurDir,'.');
 end.
