TURBO only

The Overlay Heap Size input box is where you
change the size of the overlay heap.

If you have EMS, you can decrease the size
of the overlay heap without degrading IDE
performance.

This frees more memory for compiling and
debugging your programs.

 +-----------------------------+
 | Default size: 112K          |
 | Minimum size:  64K          |
 | Maximum size: 256K          |
 | Command-line equivalent: /O |
 +-----------------------------+
