 IDE configuration file
 -----------------------
The default configuration file extension is .TC (for C or C++), .CMD
(Borland Pascal for Windows IDE), or .TP (Borland Pascal DOS IDE) . The
configuration file does not need to exist. If it does not exist, it is
created.

Note: If you delete your configuration file and run TEMC to create a new
one, all default key bindings are deleted.
