 Lo (function)
 --------------
Returns the low-order Byte of the argument.

 Declaration:
 function Lo(X): Byte;

 Target:
Windows, Real, Protected

 Remarks:
X is an expression of type Integer or Word. Lo returns the low-order byte
of X as an unsigned value.

 See Also:
 Hi
 Swap

 Sample Code:



 {Lo.PAS}

 {Sample code for the Lo function.}

 var B: Byte;
 begin
  B := Lo($1234);   { $34 }
 end.
