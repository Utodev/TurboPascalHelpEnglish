 - Run|Parameters...
 --------------------
The Run|Parameters command brings up the
Program Parameters dialog box, where you type
in command-line arguments for your running
programs exactly as if you had typed them on
the DOS command line.

You can't use DOS redirection commands (< and
>) in the Program Parameters dialog box.
