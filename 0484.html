 $W:  Windows Stack Frame Switch    (Windows only)
 ---------------------------------------------------
Generates special prolog and epilog code for far procedures and functions.

   Syntax:        {$W+} or {$W-}
   Default:       {$W+}
   Type:          Local
   Menu Command:  Options|Compiler|Windows Stack Frame

 The $W+ State:
In the $W+ state, the compiler generates special entry and exit code for far
procedures and functions.

This code enables the Windows real-mode memory manager to correctly identify
far stack frames when it adjusts the call chain after moving a code or data
segment.

 The $W- State:
In the $W- state, the compiler does not generate this special code.

Disable this option if your program will run in Windows standard mode or
Windows 386 enhanced mode only.
