 Eoln (function)
 ----------------
Returns the end-of-line status of a text file.

 Declaration:
 function Eoln [(var F: Text) ]: Boolean;

 Target:
Windows, Real, Protected

 Remarks:
F, if specified, is a text file variable. If F is omitted, the standard file
variable Input is assumed. Eoln(F) returns True if the current file position
 is at an end-of-line marker or if Eof(F) is True; otherwise, Eoln(F) returns
False.

With {$I-}, IOResult returns 0 if the operation was successful; otherwise,
it returns a nonzero error code.

 See Also:
 Eof
 SeekEoln

 Sample Code:



  {Eoln.PAS}

 { Example for Eoln }

 { For Windows: }
 { uses WinCrt; }

 begin
 { Tells program to wait for keyboard input }
   WriteLn(Eoln);
 end.
