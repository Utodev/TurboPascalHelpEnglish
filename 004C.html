 - Compile|Primary File...
 --------------------------
The Compile|Primary File command leads to the
Primary File dialog box, where you specify
which .PAS file will be compiled when you
choose Compile|Make (F9) or Compile|Build.

Use Primary File when you're working on a
program that uses several unit (.TPU, .TPP, or .TPW) or
Include files.

No matter which file you've been editing,
Make or Build always operates on your primary
file.

If you specify another file as a primary file,
but want to compile only the file in the
active Edit window, choose Compile (Alt+F9).
