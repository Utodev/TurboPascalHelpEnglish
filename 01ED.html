 +------------------------+
 | [X] Strict Var-Strings |
 +------------------------+

[X] When you check Strict Var-Strings, the
compiler compares the declared type of a
var-type string parameter with the type of
the actual parameter being passed.

If they are not identical, a compiler error
occurs.

[ ] When Strict Var-Strings is not checked
(it's disabled), no such type checking occurs.

Strict Var-Strings is equivalent to {$V}.
