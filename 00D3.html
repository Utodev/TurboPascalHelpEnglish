 - Help on the Browse Symbol dialog box
 ---------------------------------------
This dialog box consists of an input box with a history list,
OK and Cancel buttons.

 +- Symbol to browse ------------------+
 |                                     |
 +-------------------------------------+

The term at the cursor in the editor will be displayed in the input box.

You can use the history list to select a previously used term or enter a
new term into the input box.

Use the arrow keys to move around the window.
To inspect an symbol, highlight the symbol and press Enter
or by clicking directly on the symbol name.

You can then go to the source code that defines the
highlighted symbol or inspect the declaration of the
highlighted symbol.

See Using the ObjectBrowser for more details on using the ObjectBrowser.
