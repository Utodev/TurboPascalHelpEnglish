 $T:  Type-Checked Pointers Switch
 -----------------------------------
Controls the types of pointer values generated by the @ operator.

   Syntax:        {$T+} or {$T-}
   Default:       {$T-}
   Type:          Local
   Menu Command:  Options|Compiler|Typed @ Operator

 The $T- State:
In the $T- state, the result type of the @operator is always an untyped
pointer.

 The $T+ State:
When the @operator is applied to a variable referenced in the $T+ state, the
type of the result is ^T, where T is compatible only with other pointers to
the type of the variable.
