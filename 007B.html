 | Topic Search (Local menus)
 ------------------------------------
When you're in the Edit window, you can place the cursor on a
word and get Language Help on that particular word. You can
also get Language Help on the word at the cursor by:

 »  choosing Help|Topic Search, or
 »  Topic Search from the Help local
    menu, or
 »  Topic Search from the Edit local menu.

Language Help is on-line reference for many
elements of the Turbo Pascal language:
functions and procedures, reserved words,
global variables, and so on.

If the word at the cursor is not documented in
the Help system, the Help Index screen will
come up with the closest match selected.

For example, if you place the cursor on the
word "opera" and press Ctrl+F1, the Help
Index screen will come up with "operators"
selected.
