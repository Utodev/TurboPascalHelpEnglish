 - Help on the Window List dialog box
 -------------------------------------
The Window List dialog box consists of a list
box, the standard OK, Cancel, and Help
buttons, and a Delete button.

 +- Window List ------------------------+
 | C:\TURBO\EXAMPLES\TEST1.PAS          |
 | A:\MYPASCAL\OBJECT3.PAS             ±|
 | C:\TURBO\MYSOURCE\MENUS.PAS         »|
 | Output                              ±|
 | Call Stack                          ±|

The list box lists all the open windows.

 +------------+
 | [ Delete ] |
 +------------+
The Delete button removes a listed window from
the list box.
