 FindNext (procedure)     (Dos unit and WinDos unit)
 ----------------------------------------------------
Finds the next entry that matches the name and attributes specified in an
earlier call to FindFirst.

 Declaration:
 procedure FindNext(var F: TSearchRec);

 Target:
Windows, Real, Protected

 Remarks:
Errors are reported in DosError. The only possible error code is 18 (no more
files).

 See Also:
 File-attribute constants
 FileExpand
 FindFirst
 TSearchRec

 Sample Code:



 {Findfrst.PAS}

 {Sample code for the FindFirst and FindNext procedures.}

 { For Windows: }
 { uses WinDos,WinCrt; }

 uses Dos;

 var
  DirInfo: SearchRec;         { For Windows, use TSearchRec }
 begin                          { For Windows, use faArchive }
  FindFirst('*.PAS', Archive, DirInfo); { Same as DIR *.PAS }
  while DosError = 0 do
  begin
    Writeln(DirInfo.Name);
    FindNext(DirInfo);
  end;
 end.
