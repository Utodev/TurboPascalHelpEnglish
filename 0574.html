 GetDriverName (function)     (Graph unit)
 ------------------------------------------
Returns a string containing the name of the current driver

 Declaration:
 function GetDriverName: string;

 Target:
Real, Protected

 Remarks:
After a call to InitGraph, GetDriverName returns the name of the active
driver.

 Restrictions:
Must be in graphics mode.

 See Also:
 InitGraph

 Sample Code:



 {Getdrvnm.PAS}

 {Sample code for the GetDriverName function.}

 uses Graph;

 var Driver, Mode: Integer;
 begin
  Driver := Detect;   { Put in graphics mode }
  InitGraph(Driver, Mode, ' ');
  if GraphResult < 0 then
    Halt(1);
  OutText('Using driver ' + GetDriverName);
  Readln;
  CloseGraph;
 end.
