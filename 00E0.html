 +- Result ----------------------------------+
 |                                           |
 +-------------------------------------------+

The Result input box displays the value of the
expression entered in the Expression input box.

For expressions that represent a single data
element, such as i, a[i], or *p, you can
enter a new value in the New Value input box.

The debugger uses the expression's data type
to select a format for displaying the result.
You can enter format characters after the
expression to force a different format.
