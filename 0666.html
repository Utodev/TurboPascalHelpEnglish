 Graph3 Unit: 3.0 Graphics and Turtlegraphics
 ---------------------------------------------

The Graph3 unit (GRAPH3.TPU) is a direct implementation of the
turtlegraphics driver provided by Turbo Pascal 3.0.
 Go to GRAPH3 Functions and Procedures
 Go to GRAPH3 Constants

Graph3 combines Turbo Pascal 3.0's GRAPH.P and GRAPH.BIN into a single unit,
retaining the same functionality.

If you have a Turbo Pascal 3.0 program that uses the turtlegraphics driver,
the only modification you need to make is to:
  1. Remove the $I GRAPH.P compiler directive.
  2. Replace it with a reference to the Crt unit and the Graph3 unit in your
     program's uses clause.
