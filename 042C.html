 Built-in Assembler Instruction Opcodes
 ---------------------------------------
The built-in assembler supports all 8086/8087 and 80286/80287 instruction
opcodes.

These instruction opcodes are available only in the following states:

   Opcodes | State  | Meaning of State
   ================================================
   8087    | $N+    | Numeric processor enabled
   80286   | $G+    | 80286 code generation enabled
   80287   | $G+,N+ | Both enabled

For complete descriptions of the instruction opcodes, refer to your 80x86
and 80x87 reference manuals.
