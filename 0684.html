 HInstance and HPrevInst (variables)     (System unit)
 ------------------------------------------------------
HInstance contains the instance handle of the application or library as
provided by the Windows environment.

In a program, HPrevInst contains the handle of the previous instance of the
application, or 0 if there are no previous instances.

In a library, HPrevInst is always 0.
