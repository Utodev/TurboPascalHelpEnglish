 label (reserved word)
 -----------------------
A label declaration part declares labels that mark statements in the
corresponding statement part.

 Syntax:
 label identifier, ... identifier;

 Remarks:
Each label must mark only one statement.

In addition to an identifier, a digit sequence between 0 and 9999 can also
be used as a label.

Control is transferred to a label via a goto statement.

 Example:
 label 1, 2;
 goto 1
    .
    .
    .
 1:  WriteLn ('Abnormal program termination');
 2:  WriteLn ('Normal program termination');
