 TEML non-terminating commands
 -------------------------------
The non-terminating commands do not cause an exit to the IDE. They can be
used in any sequence.

 TEML non-terminating commands A - D
 TEML non-terminating commands E - H
 TEML non-terminating commands I - N
 TEML non-terminating commands O - R
 TEML non-terminating commands S - Z
