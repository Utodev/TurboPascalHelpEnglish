  Options|Tools...
 ---------------------
The Tools command displays the Tools
dialog box, where you can transfer to another program without
leaving the IDE.

You can add or delete or edit programs in the Tools menu.

You choose items from the Tools menu to run
another program without actually leaving
Turbo Pascal. (These programs you transfer
to are known as transfer programs.)

To return to Turbo Pascal from a transfer
program, you must first exit the transfer
program.

The maximum number of tools you can install is 15.
