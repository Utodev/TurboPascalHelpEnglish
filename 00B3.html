 - Help on the Read Block dialog box
 -------------------------------------
The Read Block dialog box contains an
input box, a file list, a file information
panel, and the standard OK, Cancel and
Help buttons.

This dialog box is where you copy the
contents of a file into the Edit window.

 +- Read block from ---------------+
 |                                 |
 +---------------------------------+
The Read Block From input box is where you
enter the name of the file you want to read
in, or a file-name mask.

 + Files --------------------------+
 |                                 |
 |  FILENM01.PAS  |  FILENM09.PAS  |
 |  FILENM02.PAS  |  FILENM10.PAS  |
 |  FILENM03.PAS  |  FILENM11.PAS  |
 |  FILENM04.PAS  |  FILENM12.PAS  |
 |  FILENM05.PAS  |  ..            |
 |  FILENM06.PAS  |  \MOREXAMP     |
 |  FILENM07.PAS  |  \TOURS        |
 |  FILENM08.PAS  |  \ANSWERS.DIR  |
 |  ±±±»±±±±±±±±±±±±±±±±±±±±±±±± |
 +---------------------------------+
The Files list box lists the names of files in
the current directory that match the file-name
mask in the Read Block From input box, plus
the parent directory and all subdirectories.

 +------------------------------------------+
 | C:\TP\EXAMPLES\*.PAS                     |
 | HELLO.PAS       52  May 14,1992  12:00pm |
 +------------------------------------------+
The File information panel shows the path
name, file name, date, time, and size of the
selected file.
