 - File|Change Dir...
 ---------------------
File|Change Dir brings up the Change Directory
dialog box, where you can specify a directory
to make current.

The current directory is the one Turbo Pascal
uses to save files and to look for files.

When you're using relative paths in Options|
Directories, they are relative only to this
current directory.
