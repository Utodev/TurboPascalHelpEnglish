  var (reserved word)
 ---------------------
A variable (var) declaration associates an identifier and a type with a
location in memory where values of that type can be stored.

 Syntax:
   var
     identifier, ... identifier: type;
     ...
     identifier, ... identifier: type;

 Remarks:
An absolute clause can be used to specify an absolute memory address.

The var reserved word is also used to declare variable parameters.

 Examples:
 { Variable Declarations }
   var
     X, Y, Z: real;
     I, J, K: Integer;
     Done, Error: Boolean;
     Vector: array[1..10] of real;
     Name: string[15];
     InFile, OutFile: Text;
     Letters: set of 'A'..'Z';
