 StrLIComp (function)     (Strings unit)
 ----------------------------------------
Compares two strings, up to a maximum length, without case sensitivity.

 Declaration:
 function StrLIComp(Str1, Str2: PChar; MaxLen: Word): Integer;

 Target:
Windows, Real, Protected

 Remarks:
StrLIComp compares Str1 to Str2 , up to a maximum length of MaxLen
characters, without case sensitivity.
The return value is the same as StrComp.

 See Also:
 StrComp
 StrIComp
 StrLComp

 Sample Code:



 {StrComp.PAS}

 {Sample code for the StrComp and StrlComp functions.}

 { For Windows: }
 { uses Strings, WinCrt; }

 uses Strings;

 var
  C: Integer;
  Result: PChar;
  S1, S2: array[0..79] of Char;
 begin
  Readln(S1);
  Readln(S2);
  C := StrComp(S1, S2);
  if C < 0 then Result := ' is less than ' else
    if C > 0 then Result := ' is greater than ' else
      Result := ' is equal to ';
  Writeln(S1, Result, S2);
 end.
