 - Search|Units (BP and BPW)
 ----------------------------
The Search|Units command displays a browser window with unit
information displayed.

Use the arrow keys to move around the window. To inspect a unit,
highlight the unit and press Enter or by clicking directly on the unit name.

You can then inspect all the symbols declared in the highlighted
unit's interface section.

See Using ObjectBrowser for more details on using the ObjectBrowser.
