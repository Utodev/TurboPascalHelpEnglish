02C8
 
 +---------------------------+
 | Stream registration error |
 +---------------------------+

This error is generated by the RegisterType
procedure in the Objects unit.

It indicates that one of the following errors
has occurred:

» The stream registration record does not
  reside in the data segment.
» The ObjType field of the stream
  registration record is 0.
» The type has already been registered.
» Another type with the same ObjType value
  already exists.
