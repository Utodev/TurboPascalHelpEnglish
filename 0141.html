 | [X] Desktop      |
 +------------------+

[X] When you check Auto Save Desktop, Turbo
Pascal automatically saves your desktop
settings in a TURBO.DSK, TPX.DSK or BP.DSK file whenever
you exit, and restores them when you return.

» NOTE: The desktop information will not be
saved if a .TP file is not created.

A .TP file is created automatically when Auto
Save [X] Environment is checked; it's created
manually when you choose Options|Save 
and the Desktop radio button is set to one of the following:
Current Directory or Config File Directory.

BP.EXE creates a BP.TP configuration file; TURBO.EXE creates a TURBO.TP
configuration file.
