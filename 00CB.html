 | [ Change All ] |
 +----------------+

Once you've entered the search string and the
replacement string in the input boxes, you can
choose OK or Change All to begin the search;
or choose Cancel to forget it.

If Turbo Pascal finds the specified text, it asks
if you want to make the replacement.

 » If you chose Change All, it asks you for
   every instance.

If you want to enter a search string you used
previously, press the Down arrow key to show a
history list to choose from.
