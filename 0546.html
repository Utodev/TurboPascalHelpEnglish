 DiskSize (function)     (Dos unit and WinDos unit)
 ---------------------------------------------------
Returns the total size, in bytes, of a specified disk drive.

 Declaration:
 function DiskSize(Drive: Byte): Longint;

 Target:
Windows, Real, Protected

 Remarks:
where:
   0     Default drive
   1     Drive A
   2     Drive B
   3     Drive C
etc.

 Return Value:
-1 if the drive number is invalid.

 See Also:
 DiskFree
 GetDir

 Sample Code:



 {Disksize.PAS}

 {Sample code for the DiskSize function.}

 { For Windows: }
 { uses WinDos, WinCrt; }

 uses Dos;

 begin
  Writeln(DiskSize(0) div 1024, ' Kbytes capacity');
 end.
