 Variables of Type DateTime     (Dos unit)
 ------------------------------------------
Variables of DateTime type are used in connection with the UnpackTime and
PackTime procedures to examine and construct 4-byte, packed date-and-time
values for the GetFTime, SetFTime, FindFirst, and FindNext procedures.

 type
     { Date & time recored used by PackTime }
     { and UnpackTime }
   DateTime = record
     Year,Month,Day,Hour,Min,Sec: Word;
   end;
