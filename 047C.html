 $N:  Numeric Coprocessor Switch
 ---------------------------------
Switches between the two different models of floating-point code generation
supported by the compiler.

   Syntax:        {$N+} or {$N-}
   Default:       {$N-}
   Type:          Global
   Menu Command:  Options|Compiler|80x87

 The $N- State:
In the $N- state, the compiler generates code to perform all real-type
calculations in software by calling run-time library routines.

 The $N+ State:
In the $N+ state, the compiler generates code to perform all real-type
calculations using the 80x87 numeric coprocessor and gives you access to
four additional real types: Single, Double, Extended, and Comp.

Note: You can also use the $E+ directive to emulate the 80x87. This gives
you access to the IEEE floating-point types without requiring that you
install an 80x87 chip.
