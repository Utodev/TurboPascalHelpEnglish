 ClearDevice (procedure)     (Graph unit)
 -----------------------------------------
Clears the currently selected output device and homes the current pointer.

 Declaration:
 procedure ClearDevice;

 Target:
Real, Protected

 Remarks:
ClearDevice moves the current pointer to (0, 0), clears the screen using the
background color set by SetBkColor, and prepares it for output.

 Restrictions:
Must be in graphics mode.

 See Also:
 ClearViewPort
 CloseGraph
 InitGraph
 RestoreCrtMode
 SetGraphMode

 Sample Code:



 {Clrdevc.PAS}

 {Sample code for the ClearDevice procedure.}

 uses Crt, Graph;

 var Gd, Gm: Integer;
 begin
  Gd := Detect;
  InitGraph(Gd, Gm, ' ');
  if GraphResult <> grOk then
    Halt(1);
  Randomize;
  repeat
    LineTo(Random(200), Random(200));
  until KeyPressed;
  ClearDevice;
  Readln;
  CloseGraph;
 end.
