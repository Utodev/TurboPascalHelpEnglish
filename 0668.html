 Region-Filling Procedures and Functions, 3.0 Graphics
 ------------------------------------------------------
 Procedure   | What It Does
 =========================================================================
 FillPattern | Fills a rectangular area with the current pattern using
             | the specified color
 FillScreen  | Fills entire active window with indicated color
 FillShape   | Fills an area of any shape with specified color
 Pattern     | Defines an 8x8 pattern to be used by FillPattern
