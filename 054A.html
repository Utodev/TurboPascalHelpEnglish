 DosVersion (function)     (Dos unit and WinDos unit)
 -----------------------------------------------------
Returns the DOS version number.

 Declaration:
 function DosVersion: Word;

 Target:
Windows, Real, Protected

 Remarks:
The result's low byte is the major version number, and the high byte is the
minor version number.

 Sample Code:



  {DOSVrsn.PAS}

 {Sample code for the DOSVersion function.}

 { For Windows: }
 { uses WinDos, WinCrt; }

 uses Dos;

 var
  Ver: Word;
 begin
  Ver := DosVersion;
  Writeln('This is DOS version ', Lo(Ver), '.',
 Hi(Ver));
 end.
