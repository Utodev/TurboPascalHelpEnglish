 Messages Window
 ----------------
The messages windows displays information from programs that
output through a DOS filter, for example GREP. You can scroll
through the output messages or choose to edit the program line
referenced in the message.

 Editing a program line referenced in a message:
Select the message you are interested in and double-click or press
ENTER. Your cursor will jump to the program line in the source code
that is referenced by the message in the messages window.

By default the messages window will close when your cursor goes to the
program line in the source code. If you prefer the messages window remain
open, deselect Close on Go To Source option in the Preferences dialog box.

 Tracking program lines:
Select the message you are interested in and press SPACEBAR.
If Auto Track Source is selected, referenced program lines are highlighted
in your source code automatically.
