 Where Built-in Assembler Functions Return
 ------------------------------------------
Functions using the built-in assembler directive (asm) must return their
results as follows:

   Result  | Returned in            | Comment
   =========================================================================
   Ordinal | AL (8-bit values)      | Integer, Char, Boolean, and enumerated
           | AX (16-bit values)     | types
           | DX:AX  (32-bit values) |
   Real    | DX:BX:AX               | Real type
   8087    | ST(0) on the 8087's    | Single, Double, Extended, and Comp
           | register stack         | types
   Pointer | DX:AX                  |
   String  | The temporary location |
           | pointed to by @Result  |
