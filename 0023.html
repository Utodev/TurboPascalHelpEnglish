 - File|Save As...
 ------------------
Save As opens up the Save File As dialog box,
where you can save the file in the active
Edit window under a different name, in a
different directory, or on a different drive.

You can enter the new file name, including the
drive and directory, and click or choose OK.

All windows containing this file are updated
with the new name.
