 Turbo Editor Macro Compiler (TEMC)
 -----------------------------------

TEMC.EXE is an editor macro compiler for the IDE. It processes a script
file that defines editor macros and key bindings, and produces a
configuration file that is read by the IDE to define the effects of
keyboard commands in the editor.

The script language used with TEMC is TEML.
