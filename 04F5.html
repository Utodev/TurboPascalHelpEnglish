 Sqrt (function)
 ----------------
Returns the square root of the argument.

 Declaration:
 function Sqrt(X: Real): Real;

 Target:
Windows, Real, Protected

 Remarks:
X is a real-type expression. The result is the square root of X.

 See Also:
 Sqr

 Sample Code:



 {Sqr.PAS}

 {Sample code for the Sqr and Sqrt functions.}

 { For Windows: }
 { uses WinCrt; }

 begin
   Writeln('5 squared is ', Sqr(5));
   Writeln('The square root of 2 is ',
           Sqrt(20));
 end.
