 GetPaletteSize (function)     (Graph unit)
 -------------------------------------------
Returns the size of the palette color lookup table.

 Declaration:
 function GetPaletteSize: Integer;

 Target:
Real, Protected

 Remarks:
GetPaletteSize reports how many palette entries can be set for the current
graphics mode; for example, the EGA in color mode returns a value of 16.

 Restrictions:
Must be in graphics mode.

 See Also:
 GetMaxColor
 GetPalette
 SetPalette

 Sample Code:



  {Getpltsz.PAS}


 { Example for GetPaletteSize }

 uses Graph;
 var
   Driver, Mode : Integer;
 begin
   Driver := Detect;
   InitGraph(Driver, Mode, ' ');
   if GraphResult < 0 then Halt(1);
   SetColor(1);
   OutText('Color = 1, ');
   SetColor(GetPaletteSize - 1);
   OutText('Color = Max');
   ReadLn;
   Closegraph;
 end.
