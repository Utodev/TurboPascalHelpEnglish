 CursorTo (procedure)     (WinCrt unit)
 ---------------------------------------
Moves the cursor to the given coordinates within the virtual screen.

 Declaration:
 procedure CursorTo(X, Y: Integer);

 Target:
Windows

 Remarks:
The coordinates of the upper left corner of the CRT window are (0, 0).
CursorTo sets the Cursor variable to (X, Y).

 Sample Code:



  {CursorTo.PAS}

 {Sample code for the CursorTo procedure.}

 uses WinCrt;

 var
   C: PChar;

 begin
   CursorTo(15,10);
   Writeln('Hi there.');
 end.
