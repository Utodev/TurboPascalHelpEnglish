 - How to Use Online Help
 --------------------------
(To read all of this screen, scroll down.)

You can use  Turbo  Pascal's online Help from
the Help menu or via hot keys.

 +--------+  Wherever you are in the  Turbo 
 |   F1   |  Pascal IDE, you can press F1 to
 +--------+  see a Help screen about the menu
             command or dialog-box item that's
             currently active.

in a dialog-style  Turbo  Help window:
