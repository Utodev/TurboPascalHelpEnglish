 ----------------------------
  Modify/New Tool dialog box
 ----------------------------
The Modify/New Tool dialog box is
where you add or change a program in the
Program Titles list box and the Tools
menu.

If a transfer program in the Program Titles
list box is highlighted when you select Edit
or New, the IDE automatically fills in the
input boxes in the Modify/New Tool
dialog box; otherwise, they're blank.

 +- Title ------------------------------+
 |                                      |
 +--------------------------------------+
The Title input box is where you type
in a short description of the program you're
adding.

 +- Program path -----------------------+
 |                                      |
 +--------------------------------------+
The Program Path input box is where you enter
the program name. You can include the full
path to the program, but the path is optional.

 +- Command line -----------------------+
 |                                      |
 +--------------------------------------+
The Command Line input box is where you type
in any parameters or transfer macro commands
you want passed to the program.

 +-----------------+
 | Hot Key         |
 |  ( ) Unassigned |
 |  ( ) Shift F2   |
 |  ( ) Shift F3   |

The Hot Key options assign a Shift+Function
Key shortcut to the transfer program.
