 RandSeed (variable)     (System unit)
 --------------------------------------
RandSeed stores the built-in random number generator's seed.

By assigning a specific value to RandSeed, the Random function can be made
to repeatedlly generate a specific sequence of random numbers.

This is useful for applications that deal with data encryption, statistics,
and simulations.
