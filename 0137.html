 -----------------
  Hot Key options
 -----------------
You use the Hot Key options to assign a
keyboard shortcut to the transfer program.

 +----------------+
 | ( ) Unassigned |
 | ( ) Shift F2   |
 | ( ) Shift F3   |
 | ( ) Shift F4   |
 | ( ) Shift F5   |
 | ( ) Shift F6   |
 | ( ) Shift F7   |
 | ( ) Shift F8   |
 | ( ) Shift F9   |
 | ( ) Shift F10  |
 +----------------+

Transfer shortcuts must use the Shift key
plus a function key.

Keystrokes already assigned appear in the
list but are unavailable.
