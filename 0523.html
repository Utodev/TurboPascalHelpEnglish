 Abs (function)
 ---------------
Returns the absolute value of the argument.

 Declaration:
 function Abs(X): (Same type as parameter);

 Target:
Windows, Real, Protected

 Remarks:
X is an integer-type or real-type expression. The result, of the same type
as X, is the absolute value of X.

 Sample Code:



 {Abs.PAS}


 {Sample code for the Abs function.}

 var
   r: Real;
   i: Integer;
 begin
   r := Abs(-23);       { 2.3 }
   i := Abs(-157);       { 157 }
 end.
