 ScreenSize (variable)     (WinCrt unit)
 ----------------------------------------
Determines the width and height in characters of the virtual screen within
the CRT window.

 const ScreenSize: TPoint = (X: 80; Y: 25);

The default screen size is 80 columns by 25 lines.

You can change the size of the virtual screen by assigning new values to
the X and Y coordinates of ScreenSize before the CRT window is created.

The value given by ScreenSize.X multiplied by ScreenSize.Y must not exceed
65,520.
