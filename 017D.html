TURBO only

The Editor Heap Size input box is where you
change the size of the editor heap.

An editor heap size >28K only improves IDE
performance if you're using a slow disk as a
swap device.

Don't change the default setting if you:
 » have EMS or
 » have your swap file on a RAM disk

 | Default size:  28K           |
 | Minimum size:  28K           |
 | Maximum size: 128K           |
 | Command-line equivalent: /E  |
