 +-----------------+
 | [X] Insert Mode |
 +-----------------+

[X] When you check Insert Mode, the editor
inserts text you type by pushing existing text
to the right.

[ ] When Insert Mode is not checked, any text
you type into Edit windows overwrites existing
text.

Press Ins to toggle Insert Mode when you're
working in an Edit window. (This does not
change the default setting.)
