 Seg0040 (variable)     (System unit)
 -------------------------------------
Selector for segment $0040.

 Target:
Protected; Real

 Declaration:
  var Seg0040: Word;

 Remarks:
Seg0040 contains a selector that can be used to access the ROM BIOS
workspace at segment address $0040. In real mode Seg0040 always contains
the value $0040, but in protected mode the actual value can vary.
