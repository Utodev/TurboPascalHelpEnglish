 - Run  (Alt+R)
 ---------------
The Run menu provides commands to run your
program and to start and end debugging
sessions.

Choose from any of these Run menu commands:

 Run            
 Step Over      
 Trace Into     
 Go to Cursor   
 Program Reset  
 Parameters     

To use any of the Run commands except Run|Run
and Run|Parameters, you must have compiled and
linked your program with the Options|Debugger|
Integrated check box checked.
