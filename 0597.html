 HighVideo (procedure)     (Crt unit)
 -------------------------------------
Selects high-intensity characters.

 Declaration:
 procedure procedure HighVideo;

 Target:
Real, Protected

 Remarks:
There is a Byte variable in Crt TextAttr that is used to hold the current
video attribute. HighVideo sets the high intensity bit of TextAttr's
fore-ground color, thus mapping colors 0-7 onto colors 8-15.

 See Also:
 LowVideo
 NormVideo
 TextBackground
 TextColor

 Sample Code:



 {Hivideo.PAS}

 {Sample code for the HighVideo procedure.}
 uses Crt;
 begin
  TextAttr := LightGray;
  HighVideo;   { Color is now white }
 end.
