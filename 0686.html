 PrefixSeg (variable)     (System unit)
 ---------------------------------------
In a program, the PrefixSeg variable contains the selector (segment
address) of the Program Segment Prefix (PSP) created by DOS and Windows
when the application was executed.

In a library, PrefixSeg is always 0.

For a complete description of the PSP, refer to your DOS and Windows
manuals.
