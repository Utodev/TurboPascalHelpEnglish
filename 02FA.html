 - Guidelines for Entering Directories
 --------------------------------------
Use the following guidelines when entering
directories in the Options|Directories input
boxes:

» You must separate multiple directory path
names (if allowed) with a semicolon (;).

» You can use up to a maximum of 127
characters (including whitespace).

» Whitespace before and after the semicolon
is allowed but not required.

» Relative and absolute path names are
allowed, including path names relative to
the logged position in drives other than
the current one.
