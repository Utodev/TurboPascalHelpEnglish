 set (reserved word)
 ---------------------

 Syntax:
   set of type

 Remarks:
The base type of a set must be an ordinal type with no more than 256
possible values.

The ordinal values of the upper and lower bounds of the base type must be
between 0 and 255.

A set constructor, which denotes a set-type value, is formed by writing
expressions within brackets. Each expression denotes a value of the set.

The notation [ ] denotes the empty set, which is compatible with all set
types.

 Example:
 { Set types }
   type
     Day = (Sun, Mon, Tue, Wed, Thu, Fri, Sat);
     CharSet = set of Char;
     Digits = set of 0..9;
     Days = set of Day;

 { Set constructors }

   ['0'..'9', 'A'..'Z', 'a'..'z', '_']
   [1, 5, I + 1 .. J - 1]
   [Mon..Fri]
