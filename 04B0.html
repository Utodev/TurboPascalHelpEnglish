 OvrInitEMS (procedure)     (Overlay unit)
 ------------------------------------------
Loads overlay file into EMS, if possible.

 Declaration:
 rocedure OvrInitEMS;

 Target:
Real

 Remarks:
Errors are reported in the OvrResult variable.

 See Also:
 OvrGetBuf
 OvrInit
 OvrSetBuf

 Sample Code:



 {Ovrinems.PAS}

 {Sample code for the OvrInitEMS procedure: Runs only in DOS real mode}

 uses Overlay;

 begin
  OvrInit('EDITOR.OVR');
  if OvrResult <> ovrOk then
  begin
    Writeln('Overlay manager initialization failed.');
    Halt(1);
  end;
  OvrInitEMS;
  case OvrResult of
    ovrIOError: Writeln('Overlay file I/O error.');
    ovrNoEMSDriver: Writeln('EMS driver not installed.');
    ovrNoEMSMemory: Writeln('Not enough EMS memory.');
    else Writeln('Using EMS for faster overlay swapping.');
  end;
 end.
