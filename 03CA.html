 +-------------------+
 | Too many segments |  (Protected mode and Windows only)
 +-------------------+

The executable file being generated contains more than 254 segments,
which is the upper limit of the DOS protected mode and Windows
.EXE file format.

Most likely, the preferred code segment size for the executable file is
too small. Specify a larger preferred code segment size using a
{$S segsize} compiler directive.
