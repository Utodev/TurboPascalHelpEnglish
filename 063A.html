  About Transfer Macros
 ------------------------
The Borland IDE recognizes certain macronames in the Command Line input box
(parameter string) of the Modify/New Tools Item dialog box.

There are three kinds of transfer macros:

   Kind of Macro    |   What They Do
 ====================================
   State            |   Expand to a value that's based on the state of the
                    |   IDE
   File name        |   Take file names as arguments and return various
                    |   parts of the file name. With file name macros, you
                    |   can build up new file name specifications from
                    |   existing file names.
   Instruction      |   Tell the IDE to perform an action or make a setting
