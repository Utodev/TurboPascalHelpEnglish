 GetArgCount (function)     (WinDos unit)
 -----------------------------------------
Returns the number of parameters passed to the program on the command line.

 Declaration:
 function GetArgCount: Integer

 Target:
Windows

 See Also:
 GetArgStr

 Sample Code:



 {GetArgCo.PAS}

 {Sample code for the GetArgCount function.}

 { For Windows: }
 { uses WinDos, WinCrt; }

 uses WinDos;

 var
  TheArgCount:  Integer;

 begin
  TheArgCount := GetArgCount;
  Writeln('There were ', TheArgCount,
     ' arguments to the prompt in addition to the ');
  Writeln('program name when running this program.');
 end.
