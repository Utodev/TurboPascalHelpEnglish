 Exp (function)
 ---------------
Returns the exponential of the argument.

 Declaration:
 function Exp(X: Real): Real;

 Target:
Windows, Real, Protected

 Return Value:
The value e raised to the power of X, where e is the base of the natural
logarithms.

 See Also:
 Ln

 Sample Code:



 {Exp.PAS}

 {Sample code for the Exp function.}

 { For Windows: }
 { uses WinCrt; }

 begin
   Writeln('e = ',Exp(10));
 end.
