  $CAP MSG (filter)
 --------------------
This macro captures program output into the Message window, using a DOS
filter for converting program output into Transcript window format.

A special header ensures that the file is in the correct format.

Three filters are provided:

  - TASM2MSG.EXE for Turbo Assembler (TASM) (BP and BPW)
  - GREP2MSG.EXE for GREP
  - RC2MSG.EXE for the Resource Compiler    (BP and BPW)

The source code to these filters is also provided so you can write your own
filters for other transfer programs you install.

Any program that has line-oriented messages output (file and line number)
can be used with this macro.
