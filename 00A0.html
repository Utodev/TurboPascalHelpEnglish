  Options...
 -----------
The Options command leads to the Local Browser Options dialog box,
where you set various options that control how your ObjectBrowser
works, including:

 » which symbols are browsed
 » what information is displayed and if sorting occurs

The options set in the Local Browser Options dialog box affect the
current browser window only.

The same options can be set in the Browser Options dialog box
to affect all browser windows, unless you override an option
in the current browser window with the Local Browser
Options dialog box.
