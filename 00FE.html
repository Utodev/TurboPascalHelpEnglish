 | [X] 286 Instructions |
 +----------------------+

[X] When you check 286 Instructions,  Turbo 
Pascal generates code for the 80286
instruction set.

» NOTE: Programs compiled with 80286 code
generation turned on do NOT check for the
presence of an 80286 at run time.

[ ] When 286 Instructions is not checked,
 Turbo  Pascal generates standard code.

286 Instructions is equivalent to {$G}.
