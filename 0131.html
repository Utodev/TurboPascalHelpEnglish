 +- Program Path -----------------------+
 |                                      |
 +--------------------------------------+

The Program Path input box is where you enter
the transfer program name.

You can include the full path to the program,
but the path is optional.

If you don't enter an explicit path, only
programs in the current directory or programs
in your regular DOS paths will be found.
