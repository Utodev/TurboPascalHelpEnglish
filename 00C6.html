 - Scope
 --------
You use the Scope radio buttons to specify
how much of the file you want to search in.

 +------------+
 | (.) Global |
 +------------+
Global searches the entire file, in the
direction specified by the Direction setting.
Global is the default scope.

 +-------------------+
 | ( ) Selected text |
 +-------------------+
Selected Text searches only in the block of
marked text, in the direction specified by the
Direction setting.

You can use the mouse or block commands to
mark a block of text.
