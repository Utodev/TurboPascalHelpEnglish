 - File|Open  (F3)
 ----------------
The Open command displays the Open a File
dialog box. In this dialog box, you select the
program file you want to open in an Edit
window.

The Open a File dialog box contains:

 » an input box with a history list
 » a list box to traverse
 » the standard Cancel and Help action buttons
 » Open and Replace action buttons
 » an information panel that describes the
   selected file
