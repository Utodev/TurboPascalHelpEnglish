 DosError (variable)     (WinDos unit and Dos unit)
 ---------------------------------------------------
DosError is used by many of the routines in the Dos and WinDos units to
report errors.

 var DosError: Integer;  {Error status variable}

The values stored in DosError are DOS error codes.

A value of 0 indicates no error; other possible error codes are as follows:

 Code | Meaning
 ========================
   2  | File not found
   3  | Path not found
   5  | Access denied
   6  | Invalid handle
   8  | Not enough memory
  10  | Invalid environment
  11  | Invalid format
  18  | No more files
