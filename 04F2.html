 Sound (procedure)     (Crt unit)
 ---------------------------------
Starts the internal speaker.

 Declaration:
 procedure Sound(Hz: Word);

 Target:
Real, Protected

 Remarks:
Hz specifies the frequency of the emitted sound in hertz. The speaker
continues until explicitly turned off by a call to NoSound.

 See Also:
 NoSound

 SampleCode:



  {Delay.PAS}

 {Sample code for the Delay, NoSound, and Sound  procedures.}

 uses Crt;

 begin
   Sound(220);        { Beep }
   Delay(200);        { For 200 ms }
   NoSound;           { Relief! }
 end.
