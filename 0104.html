 +----------------------+
 | [X] Overflow Checking|
 +----------------------+

When you check Overflow, the compiler
generates code to check for overflow for
the following integer operations:

  +
  -
  *
  Abs
  Sqr
  Succ
  Pred

Operations Not checked:
  Inc
  Dec

If an overflow check fails, the program
terminates and displays a run-time error
message.

Overflow checking is equivalent to {$Q+}.
