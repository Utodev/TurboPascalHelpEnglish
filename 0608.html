 name (standard directive)
 --------------------------
A name clause is included in an exports clause. A name clause consists of
the word name followed by a string constant.

 Target:
Windows, Protected

When you use a name clause, the procedure or function to be exported uses
the name specified by the string constant.

If no name clause is used, the procedure or function is exported by its
identifier and is converted to all uppercase.
