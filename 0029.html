 - File|DOS Shell
 ----------------
With the DOS Shell command, you can leave
Turbo Pascal temporarily to perform a DOS
command or enter another program.

To return to Turbo Pascal, type EXIT at the
DOS prompt, then press Enter.

Note: In dual monitor mode, the DOS command
line appears on the Turbo Pascal screen rather
than on the User screen. This allows you to
shell to DOS without disturbing the output
of your program.
