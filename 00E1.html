 +- New Value --------------------------+
 |                                      |
 +--------------------------------------+

You can enter a new value in the New Value
input box if the expression in the Expression
input box represents a single data element,
such as i, a[i], or *p.

If you modify the contents of the New Value
input box but don't press Enter, the debugger
ignores the change on exit.

The debugger uses the expression's data type
to select a format for displaying the result.
You can enter format characters after the
expression to force a different format.
