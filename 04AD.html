 OvrGetBuf (function)     (Overlay unit)
 ----------------------------------------
Returns the current size of the overlay buffer.

 Declaration:
 function OvrGetBuf: Longint;

 Target:
Real

 Remarks:
The size of the overlay buffer is set through a call to OvrSetBuf.
Initially, the overlay buffer is as small as possible, corresponding to the
size of the largest overlay. A buffer of this size is automatically
allocated when an overlaid program is executed. The initial buffer size
could be larger than 64K, because it includes both code and fix-up
information for the largest overlay.

 See Also:
 OvrInit
 OvrInitEMS
 OvrSetBuf

 Sample Code:



 {Ovrgetbf.PAS}

 {Sample code for the OvrGetBuf function: Runs only in DOS real mode}

 {$M 16384,65536,655360}
 uses Overlay;

 const ExtraSize = 49152; {48K}
 begin
  OvrInit('EDITOR.OVR');
  Writeln('Initial size of overlay buffer is ', OvrGetBuf,'bytes.');
  OvrSetBuf(OvrGetBuf+ExtraSize);
  Writeln('Overlay buffer now increased to ', OvrGetBuf,' bytes.');
 end.
