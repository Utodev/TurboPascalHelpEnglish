 - Help on the Memory Sizes Dialog Box
 --------------------------------------
The Memory Sizes dialog box consists of three
input boxes and the standard OK,
Cancel, and Help buttons.

 Real Mode Target
 +----------------------Ò---------+
 | Stack Size           º 16384   |
 | Low Heap Limit       º 0       |
 | High Heap Limit      º 655360  |
 +----------------------Ð---------+

The Stack Size input box is where you specify
the size of the stack segment (in bytes).

The Low Heap Limit input box is where you
specify the minimum required heap size (in
bytes).

The High Heap Limit input box is where you
specify the maximum amount of memory (in
bytes) to allocate to the heap.

 Protected Target (BP and BPW only)
 +----------------------Ò---------+
 | Stack size           º 16384   |
 +----------------------Ð---------+
The Stack Size input box is where you specify
the size of the stack segment (in bytes).

The Heap Size input box is where you
specify the minimum required heap size (in
bytes).

 Windows Target (BP and BPW only)
 +----------------------Ò---------+
 | Stack size           º 8192    |
 | Local Heap size      º 8192    |
 +----------------------Ð---------+
The Stack Size input box is where you specify
the size of the stack segment (in bytes).

The Heap Size input box is where you
specify the required heap size (in
bytes).
