 SegB800 (variable)     (System unit)
 -------------------------------------
Selector for segment $B800.

 Target:
Protected; Real

 Declaration:
  var SegB800: Word;

 Remarks:
SegB800 contains a selector that can be used to access the Color Graphics
Adapter video memory at segment address $B800. In real mode SegB800 always
contains the value $B800, but in protected mode the actual value can vary.
