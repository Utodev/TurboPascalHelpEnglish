 Relational Operators
 ----------------------
   Operator|Operation       |Result type|Operand types
 ==========================================================================
   =       |Equal           |Boolean    |compatible simple, pointer, set,
           |                |           |string, or packed string types
   <>      |Not equal       |Boolean    |compatible simple, pointer, set,
           |                |           |string, or packed string types
   <       |Less than       |Boolean    |compatible simple, string, packed
           |                |           |string types, or PChar
   >       |Greater than    |Boolean    |compatible simple, string, packed
           |                |           |string types, or PChar
   <=      |Less or equal   |Boolean    |compatible simple, string, packed
           |                |           |string types, or PChar
   >=      |Greater or equal|Boolean    |compatible simple, string, or packed
           |                |           |string types, or PChar
   <=      |Subset of       |Boolean    |compatible set types
   >=      |Superset of     |Boolean    |compatible set types
   in      |Member of       |Boolean    |left operand: any ordinal type T;
           |                |           |right operand: set  whose base is
           |                |           |compatible with T.
