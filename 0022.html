 - File|Save     (F2)
 ------------
The Save command saves the file that's in
the active Edit window to disk.

If the file has a default name (such as
NONAME00.PAS), Turbo Pascal opens the
Save File As dialog box so you can rename the
file and save it in a different directory or
on a different drive.
