 | Unit file format error |
 +------------------------+

The .TPU file is somehow invalid; make sure
it is a .TPU file.

The .TPU file may have been created with an obsolete version of
Turbo Pascal. In this case, a new .TPU must be created from the
corresponding .PAS file.
