 VER70 Conditional Symbol
 --------------------------
Turbo Pascal and Borland Pascal define the symbol VER70, which indicates
that you're using a Pascal compiler version compatible with Turbo Pascal
7.0.

Future versions will have corresponding predefined symbols; for example
version 8.0 would have VER80 defined, version 8.5 would have VER85 and so
on.
