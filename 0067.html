 - Options|Browser... (BP and BPW)
 ---------------------
The Browser command leads to the Browser Options
dialog box, where you set various options
that control how the ObjectBrowser works, including:

 » which symbols are browsed
 » how sub-browsing works: a new browser window or
   replace the current one
 » what is displayed and if sorting occurs
