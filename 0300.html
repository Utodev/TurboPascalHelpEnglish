 - Desktop File
 ---------------
The Desktop File options specify whether the
Turbo Pascal IDE saves your desktop to a
TURBO.DSK, TPX,DSK or BP.DSK file, and where that file resides.

Your desktop consists of Edit-window
information, window positions, history-list
contents, breakpoint locations, and other
state information.

When the IDE saves a TURBO.TP, TPX.TP, or BP.TP (configuration)
file, it also creates a TURBO.DSK, TPX.DSK or BP.DSK (desktop)
file.

 +----------------------------+
 | ( ) Current directory      |
 +----------------------------+
Current Directory directs the IDE to save
your TURBO.DSK, TPX.DSK or BP.DSK file in the current logged
directory.

 +----------------------------+
 | ( ) Config file directory  |
 +----------------------------+
Config File Directory directs the IDE to save
your TURBO.DSK, TPX.DSK or BP.DSK file in the same directory
where it saves the TURBO.TP, TPX.TP or BP.TP file.

The IDE automatically saves a desktop file
and a configuration file when you check the
Auto Save [X] Environment and [X] Desktop
options in the Preferences dialog box.

You can create a configuration file manually
with the Options|Save As dialog box.
