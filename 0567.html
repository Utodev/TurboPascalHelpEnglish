 FSearch (function)     (Dos unit)
 ----------------------------------
Searches for a file.

 Declaration:
 function FSearch(Path: PathStr; DirList: string): PathStr;

 Target:
Real, Protected

 Remarks:
Searches in a list of directories (DirList). The directories in DirList must
be separated by semicolons.
The PathStr type is defined in the Dos unit.

 See Also:
 FExpand
 FSplit
 GetEnv

 Sample Code:



  {FExpand.PAS}

 { Example for FExpand and FSearch }

 uses Dos;
 var
   S: PathStr;
 begin
   S := FSearch('TURBO.EXE',GetEnv('PATH'));
   if S = '' then
     WriteLn('TURBO.EXE not found')
   else
     { Show full path }
     WriteLn('Found as ',FExpand(S));
 end.
