 EnvCount (function)     (Dos unit)
 -----------------------------------
Returns the number of strings contained in the DOS environment.

 Declaration:
 function EnvCount: Integer;

 Target:
Real, Protected

 Remarks:
EnvCount returns the number of strings contained in the DOS environment.
Each environment string is of the form VAR=VALUE. The strings can be
examined with the EnvStr function.

 See Also:
 EnvStr
 GetEnv

 Sample Code:



 {Envcount.PAS}

 {Sample code for the EnvCount function.}

 uses Dos;

 var I: Integer;
 begin
  for I := 1 to EnvCount do
     Writeln(EnvStr(I));
 end.
