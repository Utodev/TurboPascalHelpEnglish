 - Help on the Retrieve Options Dialog Box
 ------------------------------------------
The Retrieve Options dialog box consists of
an input box, a file list box, an information
panel, and the standard OK, Cancel, and Help
buttons.

This dialog box is where you retrieve the
settings you previously saved in a .TP file
with the Save Options command.

If the Desktop File option (.DSK) in the
Preferences dialog box is set to either
Current Directory or Config File Directory,
TURBO.DSK, TPX.DSK or BP.DSK will be loaded when you load the
TURBO.TP, TPX.TP or BP.TP file.

 +- Options File Name ---------------------+
 |                                         |
 +-----------------------------------------+
The Options File Name input box is where you
type the name of the file where your Turbo
Pascal environment options are stored: the
default name for this file is TURBO.TP, TPX.TP, or BP.TP.

(The desktop options--history lists, the
desktop state, and breakpoint locations--are
stored in TURBO.DSK, TPX.DSK or BP.DSK.)

You can also type a file-name mask in this
input box, such as T*.TP.

The File list box is a standard list box. It
lists all files in the current (or specified)
directory that match the file mask given in
the Options File Name input box.

 +--------------------------------------------+
 | C:\TP6\*.TP                                |
 | TURBO.TP     846      Jul 27, 1990  5:30pm |
 +--------------------------------------------+
The information panel at the bottom of the
dialog box is standard-issue. It lists the
currently selected file by name, path and mask
(if any), size, and time-and-date stamp.
