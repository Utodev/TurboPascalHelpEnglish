 file (reserved word)
 ----------------------
A file type consists of a linear sequence of components of the component
type, which can be any type except a file type.

 Syntax:
 file of type
 OR
 file

 Remarks:
If the word of and the component type are omitted, the type denotes an
untyped file.

The predefined file type Text signifies a file containing characters
organized into lines.

 Example:
 (* File type declarations *)
 type
   Person = record
     FirstName: string[15];
     LastName : string[25];
     Address  : string[35];
   end;
   PersonFile = file of Person;
   NumberFile = file of Integer;
   SwapFile = file;
