 - Directory Tree
 -----------------
When you want to use the directory tree, press
Tab until the tree is highlighted, click the
tree, or use the hot key to get to it.

Once you're at the tree, you can press an
arrow key to select a directory or disk name.
Press Enter to show the contents of the
selected directory.

You can also double-click a directory name to
show its contents.

When you're positioned on a subdirectory,
press Enter to view the files in that
subdirectory (using the current file-name
mask).
