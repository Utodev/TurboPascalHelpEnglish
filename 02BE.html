 +----------------------+
 | Stack overflow error |
 +----------------------+

The program reports this error on entry to a
procedure or function compiled in the {$S+}
state when there is not enough stack space to
allocate the subprogram's local variables.

Increase the size of the stack by using the
$M compiler directive.

The Stack Overflow error can also be caused
by infinite recursion, or by an assembly
language procedure that doesn't maintain the
stack properly.
