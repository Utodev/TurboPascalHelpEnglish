 Copying an example from a Help file
 ------------------------------------
Help screens about procedures and functions
include example code that you can copy and
paste into an Edit window.

To copy Help example code to an Edit window:

 1) Bring up the Help screen that contains the
    example code for a particular procedure or
    function:

     » Press Ctrl-F1 when the Edit-window
       cursor is on a function or procedure.
                  OR
     » Choose Topic Search from the Help menu.
                  OR
     » Choose a procedure or function from
        the Help Index.

 2) Choose Copy Example from the Help local menu.

 3) Go to the Edit window.

 4) Choose Paste from the Edit menu.

Once you've copied the example code to an Edit
window, you can compile and run it to see how
that procedure or function works.
