 - Desktop file options
 -----------------------

BP only

The Desktop file options specify whether or not symbol
information is saved in a symbol file at the same time
a desktop file is saved.

 +--------------------+
 | ( ) Desktop only   |
 +--------------------+

If you select Desktop only, a Desktop file alone is
generated.

Your desktop information will not be saved
unless a .TP file is created. A .TP file is
created automatically when Auto Save [X] Environment
is checked; or manually with the Options|Save 
command).

 +---------------------------+
 | ( ) Desktop and symbols   |
 +---------------------------+

Check Desktop and symbols to save both the desktop
file and symbol information. Symbol information
generated during the last compilation is saved
in a symbol file (a file with a .PSM extension.)
Then the next time you start the IDE, the symbol
information generated during your last compilation
will be available to you so that you can immediately
browse and debug.
