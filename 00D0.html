 - Help on the Find Procedure dialog box
 ----------------------------------------
The Find Procedure dialog box consists of an
input box with a history list, and the
standard OK, Cancel, and Help buttons.

The way the Find Procedure dialog box works
is similar to the way the Find dialog works,
except that, with Find Procedure, you're
searching for the implementation of a procedure, or function,
rather than for instances of its use.

You must be debugging to be able to use this
command.

 +- Procedure Name ----------------------+
 |                                       |
 +---------------------------------------+
This input box is where you enter the name of
the procedure or function to search for; you can type in
the name or choose from the history list.
