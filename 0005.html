 - The Edit Window
 ------------------
Edit windows are where you type in and edit
your Turbo Pascal code. You can also do the
following in an Edit window:

 » compile your programs
 » run your programs
 » read them from disk files
 » save them to disk files

You can open as many Edit windows as you
want, but Turbo Pascal only numbers the
first nine windows (of any kind) you open.

 ----------------------------------
  Closing and Opening Edit Windows
 ----------------------------------
To close an active Edit window, click the
window's close box or choose Window|Close.

To open an Edit window, choose File|Open.
You can open the same file in more than one
window.
