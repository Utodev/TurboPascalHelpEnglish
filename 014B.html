 - Help on the Preferences Dialog Box
 -------------------------------------
The Preferences dialog box consists of three
sets of radio buttons, two check boxes,
and the standard OK, Cancel, and Help
buttons.

 +------------------+
 | Screen sizes     |
 |  ( ) 25 lines    |
 |  ( ) 43/50 lines |
 +------------------+
The Screen Size radio buttons specify whether
the Turbo Pascal environment screen displays
text in 25 lines or 43/50 lines.

 +----------------------+
 | Source tracking      |
 |  ( ) New window      |
 |  ( ) Current window  |
 +----------------------+
The Source Tracking options specify which
Edit window Turbo Pascal uses to display your
source code as you step through your program
during a debugging session.

If the line referenced isn't open in an edit window, the IDE
opens it and highlights the appropriate line

 +---------------------------+
 | Desktop file options      |
 |  [ ] Desktop only         |
 |  [X] Desktop and symbols  |
 +---------------------------+

BP only

The Desktop file options specify whether or not symbol
information is saved in a symbol file at the same time
a desktop file is saved.

Check Desktop and symbols to save both the desktop file
and symbol information generated during the last compilation.
Symbol information will then be available the next time
you start the IDE.

 +-------------------+
 | Auto save         |
 |  [ ] Editor files |
 |  [X] Environment  |
 |  [X] Desktop      |
 +-------------------+
The Auto Save options define when and how
often Turbo Pascal saves information about
your open files, settings, and options.

 +-----------------------------+
 | Desktop file                |
 |  ( ) Current directory      |
 |  ( ) Config file directory  |
 +-----------------------------+

The Desktop File set of options controls
where your desktop (in the file TURBO.DSK, TPX.DSK or BP.DSK)
is saved to on exit and restored from when you return
to the IDE.

Your desktop information will not be saved
unless a .TP file is created (automatically
or manually with the Options|Save 
command).

 +-----------------------------------------------+
 | Options                                       |
 |  [ ] Auto track source                        |
 |  [X] Close on go to source                    |
 |  [ ] Change dir on open                       |
 +-----------------------------------------------+

These options control how the browser windows and editor windows interact.
