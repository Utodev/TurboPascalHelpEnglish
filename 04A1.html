 MoveRel (procedure)     (Graph unit)
 -------------------------------------
Moves the current pointer (CP) a relative distance from its current
position.

 Declaration:
 procedure MoveRel(Dx, Dy: Integer);

 Target:
Real, Protected

 Remarks:
If the CP is at (X1,Y1), MoveRel moves it to ((X1 + Dx),(Y1 + Dy)).

 See Also:
 LineRel
 LineTo
 MoveTo

 Sample Code:



  {Moverel.PAS}

 {Sample code for the MoveRel procedure.}

 uses Graph;

 var Gd, Gm: Integer;
 begin
  Gd := Detect;
  InitGraph(Gd, Gm, ' ');
  if GraphResult <> grOk then
    Halt(1);
  MoveTo(1, 2);
  MoveRel(10, 10); { Move to the point (11, 12) }
  PutPixel(GetX, GetY, GetMaxColor);
  Readln;
  CloseGraph;
 end.
