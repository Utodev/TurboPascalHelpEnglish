 String operator
 -----------------
   Operator|Operation    |Operand types          |Result type
  ============================================================
   +       |concatenation|string type, Char type,|string type
           |             |or packed string type  |

Turbo Pascal lets you use the + operator to concatenate two string operands.

The result is compatible with any string type (but not with Char or packed
string types).

If the resulting string is longer than 255 characters, it is truncated after
character 255.
