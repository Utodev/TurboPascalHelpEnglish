 Insert (procedure)
 -------------------
Inserts a substring into a string.

 Declaration:
 procedure Insert(Source: String; var S: String; Index: Integer);

 Target:
Windows, Real, Protected

 Remarks:
Source is a string-type expression. S is a string-type variable of any
length. Index is an integer-type expression. Insert inserts Source into S at
the Indexth position. If the resulting string is longer than 255 characters,
it is truncated after the 255th character.

 See Also:
 Concat
 Copy
 Delete
 Length
 Pos

 Sample Code:



  {Insert.PAS}

 {Sample code for the Insert procedure.}

 var
   S: String;
 begin
   S := 'Honest Lincoln';
   Insert('Abe ', S, 8);  { 'Honest Abe Lincoln' }
 end.
