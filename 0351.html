 +------------------------+
 | Data segment too large |
 +------------------------+

The maximum size of a program's data segment
is 65520 bytes, including data declared by
the used units.

If you need more global data than this,
declare the larger structures as pointers,
and allocate them dynamically using the New
procedure.
