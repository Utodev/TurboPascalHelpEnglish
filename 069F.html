 InactiveTitle (variable)     (WinCrt unit)
 -------------------------------------------
Points to a null-terminated string to use when constructing the title of an
inactive CRT window.

 const InactiveTitle: PChar = '(Inactive %s)';

The string is used as the format-control parameter of a call to the Windows
WVSPrintF function. The %s specifier, if present, indicates where to insert
the existing window title.
