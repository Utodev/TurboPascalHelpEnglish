 -  Turbo  Pascal Menus
 -----------------------
The IDE (Integrated Design Environment) offers everything you
need to write, edit, compile, link, run,
manage, and debug your programs.

The menu bar at the top of the screen is the
gateway to the menus. To go to the menu bar,
press F10, or click anywhere on it.

Right click in a window to activate the local menu for that window.

You can choose from any of these commands on
the menu bar:

 File        
 Edit        
 Search      
 Run         
 Compile     
 Debug       
 Tools       
 Options     
 Window      
 Help        

 Local menus
 -----------
 Browse (BP and BPW) 
 Edit                
 Help                
 Messages            
 Watch               

For instructions on using the menu bar, go to
the Help screen How to Use the Menu Bar.

For information about menu hot keys, go to the
Help screen Hot Keys.
