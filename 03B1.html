 | Too many nested scopes |
 +------------------------+

Your program has too many nested scopes.

Each project can have no more than 512 nested
scopes, with no more than 128 nested scopes
in each module.

Each of these items counts toward the total
number of nested scopes:

 » Each unit in a uses clause
 » Each nested record type declaration
 » Each nested "with" context
