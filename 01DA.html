 - The Information Box
 ----------------------
The Compile|Information command brings up this
Information box, which displays the following
fields:

 » Statistics of the last compilation
 » Program exit code
 » Minimum and maximum heap size
 » Available memory
 » Current environment

Press Enter or click the OK button to put this
information box away.
