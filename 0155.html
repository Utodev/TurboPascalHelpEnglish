 +----------------------+
 | [ ] Overwrite Blocks |
 +----------------------+

[X] When Overwrite Blocks is on, and a block is selected, the next
letter you type replaces the selected text.

When Overwrite Blocks is on and Persistent Blocks
is off, marked blocks behave differently:

  If you do this | This happens
 =============================================
  Press Del or   | Clears entire selected text
    Backspace    |
  Insert text    | Replaces entire selected
                 | text with inserted text

The inserted text could be a character or a block
from the clipboard.
