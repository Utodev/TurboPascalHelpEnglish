  /U Command-Line Option
 ------------------------
Use the /U command-line option to specify additional directories in which to
search for units.
Separate multiple directories with semicolons.

By default, the command-line compiler first searches for units in the .TPL
file. If the units are not there, the compiler then searches for the unit
files in the current directory.
