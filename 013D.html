 - Screen Sizes
 ---------------
You use the Screen Sizes radio buttons to
choose which type of screen display the
Turbo Pascal environment will use.

 +-----------------+
 | ( ) 25 lines    |
 +-----------------+
When Screen Sizes is set to 25 Lines (the
default), Turbo Pascal uses 25 lines and 80
columns.

This is the only screen size available on
systems with a monochrome display or Color
Graphics Adapter (CGA).

 +-----------------+
 | (x) 43/50 lines |
 +-----------------+
If your PC has an EGA or a VGA, you can set
the Screen Sizes option to 43/50 Lines.

With this setting on, Turbo Pascal uses 43
lines by 80 columns for an EGA, or 50 lines
by 80 columns for a VGA.
