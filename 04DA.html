 SetColor (procedure)     (Graph unit)
 --------------------------------------
Sets the current drawing color, using the palette.

 Declaration:
 procedure SetColor(Color: Word);

 Target:
Real, Protected

 Remarks:
SetColor(5) makes the fifth color in the palette the current drawing color.
Drawing colors can range from 0 to 15, depending on the current graphics
driver and current graphics mode.

GetMaxColor returns the highest valid color for the current driver and
mode.

 Restrictions:
Must be in graphics mode.

 See Also:
 DrawPoly
 GetBkColor
 GetColor
 GetPalette
 SetAllPalette
 SetBkColor
 SetPalette

 Sample Code:



 {Getmxcol.PAS}

 {Sample code for the GetMaxColor, SetColor functions.}

 uses
  Crt, Graph;
 var
  GraphDriver, GraphMode : Integer;
 begin
  GraphDriver := Detect;
  InitGraph(GraphDriver, GraphMode, ' ');
  if GraphResult <> grOk then Halt(1);
  Randomize;
  repeat
    SetColor(Random(GetMaxColor)+1);
    LineTo(Random(GetMaxX),
           Random(GetMaxY));
  until KeyPressed;
 end.
