 Exclude (procedure)
 --------------------
Excludes an element in a set.

 Declaration:
 procedure Exclude(var S: set of T;I:T);

 Target:
Windows, Protected, Real

 Remarks:
S is a set type variable, and I is an expression of a type compatible with
the base type of S. The element given by I is excluded in the set given by
S.

The construct Exclude (S,I) corresponds to S := S + (I) but the Exclude
procedure generates more efficient code.
