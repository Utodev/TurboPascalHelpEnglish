 - File|Printer setup
 ------------------
The Printer Setup command displays the Printer Setup dialog box,
where you can specify path and command line information.

You can also check the syntax highlighting escape codes option so that
when you print, syntax elements of your code are highlighted.

Your output must go through the PRNFLTR.EXE filter program before it
prints.
