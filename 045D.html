   /F Command-Line Option
 ------------------------
The /F command-line option finds a run-time error in your source text.

When a program terminates due to a run-time error, it displays an error
code and the address (segment:offset) at which the error occurred.

To locate the error in the source text, specify that address in the /F
option:

 /Fsegment:offset

NOTE: The program and units must have been compiled with the /$D
command-line option enabled.
