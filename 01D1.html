 - Program-Management Hot Keys
 ------------------------------
(C|=Compile|, D|=Debug|, R|=Run|, S|=Search|)

  Hot Key | What It Does                     | Menu Command
 ===========================================================
  Alt+F9  | Compile                          | C|Compile
          |                                  |
  Ctrl+F2 | Resets running program           | R|Program
  Ctrl+F3 | Brings up call stack             | D|CallStack
  Ctrl+F4 | Evaluates an expression          | D|Evaluate/Modify
  Ctrl+F7 | Adds a watch expression          | D|AddWatch
  Ctrl+F8 | Sets or clears conditional       |
          | breakpoint                       |
  Ctrl+F9 | Runs program                     | R|Run
          |                                  |
  F4      | Runs program to cursor position  | R|Go To Cursor
  F7      | Executes tracing into functions  | R|Trace Into
  F8      | Executes skipping function calls | R|Step Over
  F9      | Makes (compiles, links) program  | C|Make
