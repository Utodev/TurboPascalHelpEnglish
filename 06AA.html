 WinDos Unit
 ------------
The WinDos unit implements operating system and file-handling routines.
It provides a PChar interface to the file handling functions found in the
Dos unit. Since MS-DOS requires null-terminated strings, the WinDos unit
is more efficient in communicating with MS-DOS than the Dos unit.

None of the routines in the WinDos unit are defined by standard Pascal.
