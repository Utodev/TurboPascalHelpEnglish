 - File|Print
 -------------
The Print command prints the contents of the
active Edit window.

Turbo Pascal expands tabs (replaces any tab
characters with the appropriate number of
spaces) and then sends the window contents to
the DOS print handler.

The Print command is disabled if the active
window can't be printed.

To print selected text (rather than the
window's entire contents), press Ctrl+K P.
