 begin...end construct (reserved words)
 ----------------------------------------
This construct is a compound statement. The begin and end reserved words
serve as statement brackets.

 Syntax:
 begin
   statement;
   statement;
   ...
   statement
 end;

 Remarks:
When bracketed in this way, any number of consecutive statements are treated
as a single statement.

 Example:
 (* Compound statement used within an "if" statement *)
 if First < Last then
 begin
   Temp := First;
   First := Last;
   Last := Temp;
 end;
