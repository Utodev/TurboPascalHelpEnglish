 Move (procedure)
 -----------------
Copies bytes from source to dest.

 Declaration:
 procedure Move(var Source, Dest; Count: Word);

 Target:
Windows, Real, Protected

 Remarks:
Copies a specified number of contiguous bytes (Count) from a source range
to a destination  range. No range-checking is performed.
Whenever possible, use SizeOf to determine the count.

 See Also:
 FillChar

 Sample Code:



  {Move.PAS}

 {Sample code for the Move procedure.}

 var
   A: array[1..4] of Char;
   B: Longint;
 begin
   Move(A, B, SizeOf(A)); { SizeOf = safety! }
 end.
