 Swap (function)
 ----------------
Swaps the high- and low-order bytes of the argument.

 Declaration:
 function Swap(X): (Same type as parameter);

 Target:
Windows

 Remarks:
X is an expression of type Integer or Word.

 See Also:
 Hi
 Lo

 Sample Code:



 {Swap.PAS}

 {Sample code for the Swap function.}

 var
  X: Word;
 begin
  X := Swap($1234);   { $3412 }
 end.
