  Structured types
 ------------------
A structured type holds more than one value. Structured types consist of
 array types
 file types
 object types
 record types
 set types

If a component type is structured, the resulting structured type can have
unlimited levels of structuring.

The maximum size of any structured type in Borland Pascal is 65,520 bytes.

The reserved word packed in a structured type's declaration tells the
compiler to compress data storage.
