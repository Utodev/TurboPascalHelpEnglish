This option determines what information is displayed.

 Qualified Symbols
The Qualified Symbols check box determines if you will see the inherited
symbol names fully qualified.

You must also check Inherited in the Browser Options dialog box,
or you will not see any fully qualified symbol names.

 Sort Always
By default, the ObjectBrowser displays identifiers in the Scope view
in the order they are declared.

If you prefer to see them listed in alphabetical order,
check the Sort Always option.
