 + Filter Path ------------------------+
 |                                     |
 +-------------------------------------+

The Filter Path input box allows you to specify a path for printer
filter information. The default setting is PRNFLTR.

You can indicate an Epson, HP LaserJet or PostScript printer.

 Default:
 $NOSWAP /ASCII

 Epson:
 $NOSWAP /EPSON

 HP LaserJet:
 $NOSWAP /HP

 PostScript:
 $NOSWAP /PS

 /Lxxx        - Lines per page (default = 55)
 /Txx         - Tab size (default = 8)
 /O[filename] - Output to file or device (default = LPT1)

If you have a different kind of printer, you can modify PRNFLTR.PAS
to send the codes your printer accepts.

To print to a text file, use the syntax:
 /O<filename> (no space after the /O)
