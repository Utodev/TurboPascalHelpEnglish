 WindowSize (variable)     (WinCrt unit)
 ----------------------------------------
Determines the initial size of the CRT window.

  const WindowSize: TPoint = (X: cw_UseDefault; Y: cw_UseDefault);

The default size allows Windows to select a suitable size for the CRT
window.

You can change the initial size by assigning new values to the X and Y
coordinates before the CRT window is created.
