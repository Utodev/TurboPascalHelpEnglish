 +- Options file name ---------------------+
 |                                         |
 +-----------------------------------------+

The Options File Name input box is where you
type the name of the file where your Turbo
Pascal environment options will be stored:
the default name for this file is TURBO.TP, TPX.TP or BP.TP.

(The desktop options--history lists, the
desktop state, and breakpoint locations--are
stored in TURBO.DSK, TPX.DSK or BP.DSK.)

Once you've typed a file name in this input
box, press Enter (or choose OK) to save that
file.

You can also type a file-name mask in the
input box, then choose the file you want from
the Files list box.
