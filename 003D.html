 - Search|Objects (BP and BPW)
 ------------------------------
The Search Objects command displays a browser window which displays
all of the objects in your application, arranged as a horizontal "tree" to
show ancestor-descendant relationships.

You can highlight any object in the display by using the arrow keys,
or by clicking directly on the object name.

You can then go to the source code that defines the highlighted object
or inspect the fields and methods of the highlighted object.

See Using ObjectBrowser for more details on using the ObjectBrowser.
