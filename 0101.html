 +--------------------+
 | [X] Range Checking |
 +--------------------+

[X] When you check Range Checking, the
compiler generates code to check that array
and string subscripts are within bounds, and
that assignments to scalar-type variables
don't exceed their defined ranges.

If the check fails, the program halts with a
run-time error.

[ ] When Range Checking is unchecked, no such
code is generated.

Range Checking is equivalent to {$R}.
