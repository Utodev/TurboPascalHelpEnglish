 Sin (function)
 ---------------
Returns the sine of the argument.

 Declaration:
 function Sin(X: Real): Real;

 Target:
Windows

 Remarks:
X is a real-type expression. Returns the sine of the angle X in radians.

 See Also:
 ArcTan
 TypeOf
 Cos

 Sample Code:



 {Sin.PAS}

 {Sample code for the Sin function.}

 { For Windows: }
 { uses WinCrt; }

 var
  R: Real;
 begin
  R := Sin(Pi);
  Writeln ('The Sin of Pi is', R);
  Readln;
 end.
