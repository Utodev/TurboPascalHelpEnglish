 qualified method activation
 ----------------------------
Within a method, a function call or procedure statement allows a qualified
method designator to denote activation of a specific method. This type of
call is known as a qualified method activation.

The object type specified in a qualified method designator must be the same
as the enclosing method's object type or an ancestor of it.

Inherited can be used to denote the immediate ancestor of the enclosing
method's object type.
