 WindMin and WindMax (variables)     (Crt unit)
 -----------------------------------------------
Stores the screen coordinates of the current window

These variables are set by calls to the Window procedure.

WindMin defines the upper left corner.
WindMax defines the lower right corner.

The X coordinate is stored in the low byte, and the Y coordinate is stored
in the high byte.

For example, Lo(WindMin) produces the X coordinate of the left edge, and
Hi(WindMax) produces the Y coordinate of the bottom edge.

The screen's upper left corner corresponds to (X,Y) = (0,0).

 Note:
For coordinates passed to Window and GotoXY, the upper left corner is at
(1,1).
