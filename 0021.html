 - File|New
 ----------------
New opens a new Edit window with the default
name NONAMExx.PAS (the xx stands for a number
from 00 to 99) and automatically makes the
new Edit window active.

These NONAME files are used as a temporary
edit buffer; Turbo Pascal prompts you to
name a NONAME file when you save it.
