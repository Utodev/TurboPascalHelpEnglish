  The Edit Local menu
 ----------------------
The Edit local menu is displayed when you right click in the
browser window or title bar.

The following options are displayed:

  Cut                    
  Copy                   
  Paste                  
  Clear                  
  Open file at cursor    
  Browse symbol at cursor
  Topic Search           
  Toggle breakpoint      
  Go to cursor           
  Evaluate/Modify...     
  Add Watch...           
  Options...
