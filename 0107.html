 +---------------------+
 | [X] Extended Syntax |
 +---------------------+

[X] When you check Extended Syntax,  Turbo 
Pascal's syntax is extended so you can use
user-defined function calls as statements,
as if they were procedures.

Extended syntax also enables you to use null-terminated strings.

[ ] When Extended Syntax is not checked (it's
disabled), you can't use user-defined function
calls as statements, or null-terminated strings.
Extended Syntax is equivalent to $X.
